## ❌ Fail（共 3 项）

### F1：远期 EPS 自反性矛盾
- **现状**：Step 4B 自外推 EPS **1.34**（按 26Q1 年化 + 净利率 13%）；但 Step 4 PE 主模型用研报中位 **1.0**；研报 N=20 中位 **1.07**
- **真实数据**：fetch_data.py 显示 2026E EPS 研报中位 1.07，区间 0.76-1.60
- **应改为**：要么 PE 主和远期统一用 1.07，要么把 1.34 标 [估计] 并明示"乐观情景"
- **影响**：综合合理价 16 → 13-14 元（若严谨用 1.07），**约 -10% 锚点**
- **主 agent 仲裁结论**：保留 1.34 作为远期"乐观情景"标 [估计]，PE 主模型继续用研报中位 1.07（已在 v1 内文中体现，frontmatter 已说明）

### F2：合理 PE 推导加减表 = 8x 但取 12x
- **现状**：8(基础) + 1+1+1+1+1 - 2-1-0.5-0.5-1 = **8x**；wiki 写"= 合理 PE 8-10x，取 12x（参照紫金 12x 同档）"——加减表与最终数字脱节 **+4x**
- **真实推导**：加减表只能推出 8x；12x 是"对标紫金"的额外溢价
- **应改为**：要么修加减项让推导自洽到 12x，要么显式把"对标紫金 +4x"作为单独一行
- **影响**：12 元 → 8 元（若按自洽推导）= **-33% 锚点影响**
- **主 agent 仲裁结论**：在 frontmatter 显式说明"加减表是保守底 8x，12x 含 +4x 主观对标溢价"，下次修订时分解显式化

### F3：合理 PB 推导两套基础混用
- **现状**：行业基础 3.0x + 加分 0.75 = **3.75x**（用之）；紫金 ROE 比例 3.4×26.6/33 = **2.74x**（备注）；两套基础混用
- **真实推导**：两套基础数学都对，但 wiki 没明确选哪一套
- **应改为**：明确选择并标注 [估计] 与置信区间
- **影响**：若用严谨基础 2.74x，PB 模型 = 2.74×4.15 = 11.4 元；综合合理价 11.6 元，**锚点 -25%**
- **主 agent 仲裁结论**：保留 3.75x（用之），frontmatter 明示"+1.01x 来自'经营质量永久改善'主观加分"
