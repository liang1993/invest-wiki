# Stock Deep-Dive 框架学习沉淀

> 本文件只记录**产品/经营/竞争/战略层面**的方法学。
> **财务/估值方法**（单位经济、扣非桥接、CapEx 拐点、汇兑归因等）一律不进——那是 value-invest 的边界。
>
> 每次 deep-dive 完成后，把"我们框架原本没有的产品分析方法/维度"沉淀到这里。
> Step 1 必读本文件——历史化石如果没人读 = 不存在。
>
> 格式：每条记录在对应维度小节下追加 `- YYYY-MM-DD <标的> | <一句话方法> | 案例：<具体产品/竞争/战略发现>`
>
> **固化判断**：单条停留在本文件（案例库）；同一维度积累 ≥2 个不同标的 → 候选纳入下一版 SKILL.md 主流程；与现有 SKILL.md 维度重复 → 删除。

## 产品 SKU 矩阵还原（拆口径找伪繁荣）

> 把"高端化销量 +37%"这种汇总数据，还原到具体 SKU 层面——爆款是谁、滞销是谁、价格带怎么分布、对位什么竞品。

- 2026-04-29 比亚迪 | "高端化销量 7.7 万 +37%" 拆 SKU 后真核心高端腰斩 | 案例：腾势/仰望/方程豹合计 7.7 万辆 +37.3%，剔除方程豹 5（17.98-21.98 万定价 / 硬派越野 / 对位坦克 300）+ 钛 7（~25 万 / 对位智己 LS6）+ 钛 3（A0 级 ~10 万 / 对位海鸥）后，真核心高端（30 万+ 仰望/腾势）仅 3.2 万辆环比腰斩（来源：海豚君，verified: partial）
- **通用模式**（可在任何行业复用）：
  - "高端化占比" → 拆"伪高端"（下沉爆款定价高端品牌） vs 真核心高端
  - "出海占比" → 拆"自有品牌出口 vs 散件 KD vs 代工"
  - "市占率提升" → 看绝对销量是否增长，还是行业整体萎缩下的被动提升
  - "新业务收入 +X%" → 看绝对值占比是否真的有意义

**固化建议**：已纳入 SKILL.md Step 2 强制拆口径。

## 竞品对位的具象化（不是泛泛"行业内卷"）

> 写"行业竞争加剧"是废话。要写具体到对手公司的具体产品名 + 具体打法 + 具体价格带。

- 2026-04-29 比亚迪 | 国内主流价格带（10-20 万）受吉利银河"鲸鱼策略"具体侵蚀 | 案例：吉利银河 L6 / 星愿 等产品凭借"高配置+强性价比"在 10-15 万插混带切走份额，导致比亚迪秦 PLUS 128km 版插混市占率压力（来源：海豚君，verified: partial）
- 2026-04-29 比亚迪 | 海外市场对位：东南亚/南美 vs 长城（俄罗斯/巴西）vs 上汽名爵（欧洲）| 案例：比亚迪海外结构以东南亚（泰国/印尼）+ 南美（巴西）为主，欧洲占比小，所以欧盟 17% 反补贴税冲击较小；vs 长城海外以俄罗斯/中亚为主（避开欧美）；vs 上汽名爵欧洲销量大受关税重创（自评 ⚠️待 WebSearch 国别销量验证）
- **通用规则**：
  - 写竞争前必须列 ≥ 2 家具体对手公司 + ≥ 2 款具体竞品
  - 海外业务必须按地理 cluster 拆（北美 / 欧洲 / 东南亚 / 南美 / 中东 / 中亚），不同 cluster 政策风险/竞争对手完全不同
  - 价格带分析按 5 万元一档分（< 10 / 10-15 / 15-20 / 20-25 / 25-30 / 30+），同档内才有真竞品

## 政策对冲性质判断（同一政策对不同标的影响相反）

> 同一政策对竞争位置不同的公司影响可能相反，不能照搬一家分析到另一家。

- 2026-04-29 双标的对比 | 反内卷政策对比亚迪 vs 长城 | 案例：反内卷限价对比亚迪是 -2x 减分（限制其降价反击的工具被夺走），但对长城是中性偏正面（限制对手降价空间，长城本就不是价格战发起者）。**判断方法**：先识别该公司在行业中是"价格战发起者 / 跟随者 / 被压制者"，再判政策影响方向（自评 + 光大证券 verified）
- **通用规则**：
  - 写政策风险时必须问"对该公司是 +/-/中性"
  - 不能直接套用行业平均影响（行业 -2x 不代表本公司 -2x）
  - 明确该公司在该政策下的"被动 vs 主动"位置（如反内卷下，比亚迪被动 / 长城主动）

## 管理层战略意图的反向解读（产品/战略层面）

> 公司公告/电话会议给的口径常常是粉饰过的，要从**反向行为**推断真实战略判断。本节只记录**产品/战略层面**的反向信号，**财务费用率方向归 value-invest 远期分析（Step 4B）**。

- 2026-04-29 双标的对比 | 研发投入方向（定性二元判断：加注 / 抽身）| 案例：比亚迪 Q1 研发**主动收缩**暗示管理层对全年悲观；长城 Q1 研发**刚性加注**显示管理层对 H2-2027 仍有信心（来源：光大证券 + 新浪 Q1 verified；具体百分比归 value-invest）
- **通用规则**：
  - 研发投入方向（加注 / 抽身 / 持平）= 管理层对未来 12-24 个月资源配置信心的代理，**只做定性二元判断**，不引用百分比
  - 高端化销售口径变化（如新增"伪高端"品牌划分）= 管理层试图美化数字
  - 股东信 / 股东会发言 vs 实际产品上市节奏的差距 = 战略落地度
  - 经销商/合作伙伴变动（如核心经销商退出 / 渠道整合公告）= 渠道战略信号

**适用扩展**：互联网（AI 投入方向）、半导体（产能扩张方向）、医药 CXO（订单结构变化）

> ⚠️ **已删除条目**：原 "管理层目标 Q1 完成度 = 全年达标概率代理"（基于"100 亿净利目标 Q1 完成 9.45%"的利润预测）已移除——本质是利润预测，归 value-invest 远期分析（Step 4B 多情景假设 / 含息年化回报率）。本文件不收纯财务方法。

## 市占率结构性下滑 vs 周期波动（识别真危机）

> 营收下行可能是周期波动也可能是结构性危机。判断方法：看同行业份额变化方向。

- 2026-04-29 格力 | 营收 -10% 是结构性危机不是周期波动 | 案例：格力 2025 营收 1,711 亿（-9.96%）看似可能是周期，但同行业份额变化揭示真相——美的 29%（-1pp 仅小幅下行）/ 海尔 15% **(+4pp 唯一净增长)** / 格力 17% **(-2pp 同行最大下滑)** / 小米 15.32% 线上（+4.78pp 新进入者飙涨）。**当同行业有公司份额净增长时，下行公司是结构性失血，不是周期共振**（来源：白电三巨头 2025 战局，partial）
- **通用规则**：
  - 看到"营收下行"先问"同行同期份额变化"
  - 同行业一家增 + 一家平 + 一家降 → 降的那家是**结构性失败**
  - 同行业全部下行 → 真周期共振（如地产链 2024-2025）
  - **结构性失败的合理 PE 应低于周期失败**（前者预期不会自然修复）

## 用户口碑反差作为份额下滑先行指标

> 官方调研（满意度第一）和真实投诉（黑猫数量）反差是结构性份额下滑的微观证据。

- 2026-04-29 格力 | 官方调研连续 11 年顾客满意度第一 vs 黑猫投诉 9,384 件（vs 美的 7,616 / 海尔 2,659）| 案例：按市占率折算，**格力投诉密度（投诉/份额）是海尔的 3.1 倍**——官方调研是抽样滞后数据，真实投诉是用户主动反馈。两者反差能在份额下滑显现前 6-12 个月预警（partial）
- **通用规则**：
  - 比较"官方满意度调研" vs "投诉量 / 市占率密度"
  - 反差比 > 2 倍 → 警示信号（实际体验恶化但官方数据没反映）
  - 投诉密度行业最高 + 份额下行同步发生 → 强结构性恶化信号

## 个人 IP 深度绑定渠道（治理风险）

> 当公司渠道变革深度绑定 CEO/创始人个人 IP（如"董明珠健康家"），接班人危机会同时蚕食渠道资产。

- 2026-04-29 格力 | "董明珠健康家"2025 启动，目标 10,000 家替代 2 万家专卖店 | 案例：渠道扁平化叙事完整，但 71 岁董明珠连任董事长 + 王自如 2024-08 离职 + 接班人方案 5 年不明的背景下，**个人 IP 渠道资产 = 接班人风险的放大器**——一旦董明珠离任，"健康家"品牌可能失去人格化支撑（partial）
- **通用规则**：
  - 渠道资产是否绑定 CEO/创始人 IP？（如雷军/董明珠/Steve Jobs/Elon Musk）
  - 如绑定，接班人不明对该公司估值的负面影响应放大（PE 减 1-2x）
  - 反向：去人格化的渠道资产（如美的渠道、丰田渠道）治理风险溢价低

## 用户口碑作为产品力先行指标

> 销量数据是滞后的，用户口碑（小红书/微博/汽车之家论坛/B 站测评/抖音热度）是先行 6-12 个月的信号。

- （待积累首例）：尚无标的做过用户口碑维度。**未来 deep-dive 应主动 WebSearch**：
  - 小红书"<品牌名> + 真实车主" 关键词
  - 汽车之家 / 懂车帝论坛投诉与晒单 ratio
  - 抖音视频热度（车评 KOL 评测）
  - B 站车评频道（如蛋蛋的车 / 吴佩频道）
- **通用规则**：销量见顶/见底**前** 6-12 个月，往往伴随口碑结构性变化。例如理想 ONE 销量见顶前 6 个月，论坛已经出现"二代大屏怎么这么卡"的批量吐槽

## "细分龙头" 假设的反向校核（市占率 vs 单款销量）

> 写"细分龙头 X% 市占率"时，必须配套问"龙头单款销量是否被反超"。市占率是分子分母游戏（行业整体萎缩可能伪造市占率提升），单款被反超是更强的危险信号。

- 2026-04-29 长城汽车 | "硬派越野细分龙头 47%" 假设动摇 | 案例：wiki SOTP 给坦克 1.5x P/S "国内高端品牌溢价" 基于"坦克 300 = 不可替代细分龙头"假设，但 2025 年 1-11 月**方程豹豹 5（17.98-21.98 万定价）4.78 万首次反超坦克 300 4.58 万**（新浪 / partial），BJ40 车系（增程+燃油合计 6.69 万）也已超过坦克 300 单款。**反向校核方法**：写"细分龙头 X%" → 问"龙头单款 vs 第二名差距是扩大还是缩小" → 若差距缩小到 10% 以内或被反超，估值溢价假设触发重审
- **通用规则**：
  - 任何"细分龙头 X% 市占率" 必须配套查"龙头单款 vs 第二名销量差距"
  - 单款被反超 = SOTP 高端分部 P/S 假设触发重审
  - 注意"伪高端定价下沉"打法：方程豹 5（17.98 万对位坦克 300 22-25 万）通过定价下沉切走份额，是 2024-2026 国内"以低打高"通用打法

## 海外业务的"国别 cluster + 政策环境"双重拆解

> 海外业务不能只看"占比 X%"汇总，必须按地理 cluster 拆，每个 cluster 政策环境/竞争对手/产品需求完全不同。

- 2026-04-29 长城汽车 | Q1 海外销量 +43.1% 应拆国别看 | 案例：长城海外结构以俄罗斯（受益西方车企退出）+ 巴西（本土工厂建产中）+ 中亚 + 中东为主，**欧洲销量基数小**（受 17% 反补贴税但影响有限）；vs 比亚迪以东南亚 + 南美为主；vs 上汽以欧洲为主受关税重创（自评 ⚠️ 长城海外国别精确拆分待 WebSearch verified）
- 2026-04-29 长城汽车 v2 deep-dive | 海外占比 +22pp 掩盖了"俄罗斯单一市场依赖"脆弱性 | 案例：俄罗斯 2024 全年 22.98 万占海外 ~50%（单一市场依赖），但 H1-2025 哈弗在俄 -21.5%（俄全市场 -27.6%）；2025 全年海外 +11.7% 完全靠拉美（巴西 +20%，但 8 月才投产 5 万产能）+ 澳洲（4.28 万稳态）+ 中东补足俄缺口。**结论**：海外汇总数字健康但结构脆弱——单一最大市场下滑 20%+ 仍能保持总量 +11.7% 增长，依赖的是"小市场 +30% 大市场 -20%"的算术游戏，质量低于纯增长的健康海外故事。**通用规则**：海外 > 30% 占比的标的，必须查"最大单国市场 YoY"——若主市场 -20%+ 同时总海外 +10%+，叙事质量打折
- **通用规则**：
  - 海外销量 > 20% 占比的标的，必须按地理 cluster 拆（≥ 4 个 cluster）
  - 每个 cluster 标注：政策环境（关税/反补贴/本土化要求）、主要竞品、销量趋势
  - 不能用"海外 +X%"一句话带过

## 产品周期与新品节奏的 12-24 月窗口

> 短期销量受限于现售产品，未来 12-24 个月销量增量靠新品。必须 WebSearch 公司未来 6-12 个月的产品上市 timeline。

- 2026-04-29 比亚迪 | 2026 H1 - H2 产品/技术 timeline | 案例：第二代刀片电池 + 兆瓦级闪充（2026/3/5 发布，5 分钟充电 10-70%）/ 腾势 Z9GT（1036 公里续航）/ 6C 闪充技术下放至 15-20 万元车型 / 闪充网络计划 2026 年底 2 万座 / 天神之眼 5.0 智驾下沉至 10 万级（来源：海豚君，verified: partial）
- 2026-04-29 长城汽车 | 2026 H1-H2 产品 timeline 信息真空 | 案例：三份外部点评均未提及长城新车型/新技术节点，需 WebSearch 公司公告补充（待核实）
- **通用规则**：
  - 每次 deep-dive 必须建立"未来 12-24 个月新品 timeline 表"
  - 信息真空本身就是信号——产品节奏不透明 = 投资者预期不充分 = 估值假设承担更大不确定性
  - 技术驱动行业（汽车/半导体/电池/医药）必须 WebSearch 公司官网+公告，不能只读财报和卖方点评

## IP / 单一爆款集中度的反向校核

> 对 IP 平台型公司（潮玩/玩具/影视/游戏），写"占比 38%"时必须配套问"Top 2 IP 体量是否达 Top 1 的 ≥30%"——这是接棒能力的代理指标。

- 2026-05-07 泡泡玛特 | THE MONSTERS（Labubu）141.6 亿占 38.1%，SKULLPANDA 35.4 亿 = Labubu 的 25%（**接近警戒线 30% 但未达**）；星星人 1.2→20.6 亿(+1617%) 证明流水线孵化能力可复用，但量级仍是 Labubu 的 1/7；**风险节点**：Labubu 半衰期 < 12 个月 + 无 IP 跨过百亿 → 单 IP 估值溢价崩塌
- **通用规则**：
  - IP 集中度 > 30% 警示，> 40% 触发风险溢价
  - Top 2 / Top 1 体量比 < 30% 时 = 接棒断档；30-50% = 健康；> 50% 是分散平台
  - 看公司是否有"流水线孵化机制"证据（多个新 IP 跨 5 亿 / 10 亿门槛 = 平台叙事；只有 Top 1 撑天 = 单点风险）
  - Sanrio 数十年只有 Hello Kitty 一条线 vs 泡泡玛特已有 6+ 个 5 亿+ 头部 IP——平台度高于 Sanrio 但需观察持续性

## 单店产出 vs 门店密度曲线（连锁 / 零售业务通用）

> 海外稀缺性溢价是常态（2-3x 国内单店产出），但门店密度增至 X 倍后必然回落。需建立"密度 vs 单店产出"曲线模型，避免线性外推。

- 2026-05-07 泡泡玛特 | 海外单店 1.06 亿 vs 国内单店 0.47 亿（**2.3x 稀缺性溢价**）；美国 64 家 → 2026 年计划 100+ 家 → 长期可能 200-500 家；**关键问题**：单店产出回落至 1.0 / 0.7 / 0.5 亿 三档情景对中长期 P/S 假设的影响截然不同
- **通用规则**：
  - 海外/新市场门店稀缺时单店产出 = 国内 2-3x（旗舰效应 + 客单升级）
  - 门店密度饱和后必然向国内单店产出收敛
  - 估值假设必须做"密度阶段"敏感性，不能线性外推
  - 适用：连锁餐饮（蜜雪/星巴克）、潮玩零售、电动车直营店、奢侈品

## 二级市场 / 二手交易破发作为高溢价资产先行指标

> 对收藏属性 + 黄牛溢价的资产（潮玩/球鞋/盲盒/限量品），二手价跌破官方价是稀缺性削弱的早期信号，领先销量数据 6-12 个月。

- 2026-05-07 泡泡玛特 | 2026/1/4 起 Labubu 3.0 端盒二手价跌破官方价（新浪财经 partial），三代隐藏款"本我"暴跌 88% ⚠️待原始数据源溯源；公司应对策略"放量任买"压抑黄牛但同时削弱稀缺性 — **短期销量正面，中期对 Labubu 高溢价定位负面**
- **通用规则**：
  - 看到"二手破发"信号 = 高溢价资产稀缺性松动早期
  - 公司"任买放量"应对 = 主动牺牲稀缺性换取销量，是 IP 生命周期由"爆发→平台"转折信号
  - 监测渠道：闲鱼/千岛/StockX/eBay 端盒/隐藏款溢价比
  - 适用：潮玩、球鞋、限量奢侈品、邮票/纪念币（虚拟资产 NFT 同理）

## 单一 cluster 驱动整体估值溢价（汇总占比掩盖结构）

> 海外/新业务汇总占比看上去合理，但深入拆 cluster/产品后发现：单一 cluster 才是估值溢价驱动，其余 cluster 仅贡献体量。

- 2026-05-07 思源电气 | "海外占比 27%" 汇总掩盖**单 cluster（北美 AIDC）的 50x PE 估值溢价驱动** | 案例：海外 58.03 亿（占比 26.94%）按 cluster 拆——欧洲/中东/南美/东南亚四 cluster 共 ~55 亿（贡献体量），北美 cluster 仅 2-3 亿订单但带来 50x PE 整体溢价（相对国资同业 25x 高 1 倍）。AIDC 稀缺性叙事 = 单一小 cluster 驱动整体估值跳升
- **通用规则**：
  - 海外/新业务带来估值溢价时，必须问"哪一个 cluster 真正驱动溢价"
  - 单一 cluster 占海外 < 5% 但带来整体 30%+ 估值溢价 = **高风险故事**（兑现节奏拖累概率高）
  - 处理方法：用此 cluster 的"项目级空间 / 当期实际订单"比值评估兑现风险
  - 适用：汽车（北美 vs 东南亚）、工业（AIDC vs 主网）、医药（FDA vs 国内）

## 国资 vs 民营龙头估值溢价的三因子分解

> 在国资主导行业（电力/通信/航天/部分医药），民营龙头通常享有显著估值溢价，但溢价应与基本面三因子绑定，不能纯叙事估值。

- 2026-05-07 思源电气 | 民营龙头估值溢价 = ROE + 决策效率 + 出海积极度 三因子叠加 | 案例：电力设备同业 ROE 思源 22.6%（民营）vs 国电南瑞 16.3% / 平高 10.3% / 特变 8.8%（国资）——**民营 ROE 比国资龙头高 6-14pct 是结构性溢价基础**；2026Q1 营收增速思源 +41.6% vs 同业 -2.4%~+7.5%（民营出海积极度差距）；估值 PE 思源 50x vs 南瑞 25x vs 特变 13x（溢价 1.5-3x）
- **通用规则**：
  - 国资主导行业的民营龙头估值溢价 = (a) ROE 优势 (b) 决策效率（季度增速差）(c) 出海积极度，三因子各 +0.5x，三者均显著 = 1.5-2x 溢价
  - 仅有 1 个因子显著（如出海但 ROE 平庸）= 1.2-1.3x 溢价（如部分出海二线民营）
  - 三因子均不显著（如某些细分赛道民营）= 不享溢价（PE 等同国资）
  - 适用：电力（思源 vs 南瑞）、通信、汽车（比亚迪 vs 上汽）、化工（万华 vs 中石化）

## 银行业 SKU 矩阵的适配方法

> 银行没有车型那种离散 SKU，但贷款产品 / 中收业务 / 客户层级也可以拆成"伪 SKU"。

- 2026-05-08 平安银行 | 银行业 deep-dive 适配 SKU 矩阵 = 四层结构 | 案例：(1) **对公条线**：战略客户 / 科技 / 普惠 / 供应链 / 跨境 / 房地产；(2) **零售条线**：房贷 / 消费贷 / 经营贷 / 信用卡 / 汽车金融；(3) **中收业务**：代理保险 / 基金 / 理财；(4) **客户底盘**：零售客户 / 财富客户 / 私行客户 / MAU。每层区分"爆款 +10% 以上 / 稳态 0-5% / 滞销 < 0% / 出清接近尾声"。平银 Q1 数据：对公爆款（战略客户 +14%）+ 零售出清完成（信用卡 5000→4020 企稳）+ 财富爆发（+55.1% 但是低基数反弹）
- **通用规则**：
  - 银行业拆条线必须区分"做精 / 做强 / 做专"战略主轴 vs 配套业务
  - 对公条线看头部客户（战略客户）+ 政策红利（普惠 / 科技）+ 历史包袱（地产）三条线
  - 零售条线必须问"高风险出清是否完成"——出清中 vs 出清完成 vs 重新扩张三个阶段对估值含义完全不同
  - 中收业务必须做与同业对位（招行 vs 兴业 vs 平银），平银财富管理 +55% 是 Q1 个例，年化合理 +20-25%
  - 客户底盘看四维：户数 / 增速 / AUM / MAU——全面落后 2-3x = 不同档

## "出清完成" vs "新风险形成" 双线并行

> 改革型公司常被叙事简化为"出清完成 = 重回增长"，但 deep-dive 应主动检查老问题完成的同时是否有新问题正在形成。

- 2026-05-08 平安银行 | 高风险零售出清完成 + 逾期偏离度上行 双线并行 | 案例：(a) 老问题完成证据——新一贷 1500→100 亿（出清 93%）、信用卡 5000→4020 亿（已企稳）、零售不良生成连降 6 季度；(b) 新问题形成证据——逾期 60 天偏离度 0.67→0.86、可疑类 +12.1%、关注贷款 +3.9%——意味"逾期增长快于不良认定"，第二轮潜在不良正在形成（partial）
- **通用规则**：
  - 看到"出清完成 / 改革兑现"叙事时，必须同步检查 3 类前置指标：(1) 逾期偏离度 / 关注贷款（银行）；(2) 应收账款周转天数 / 存货跌价（制造业）；(3) 存量门店单店产出 / 复购率（零售连锁）
  - 改革成效（确定性强）+ 新风险形成（不确定性）= 估值不应像市场那样简单上修，应保留下行情景的可信度
  - 反例：2014-2015 民生银行小微"完成出清"叙事兑现后随即遭遇第二轮不良暴雷

## 同业对位的能级判断（"X 平替"叙事的反向校核）

> 当 PB 折价反映"龙头 vs 二线"时，市场常给二线"龙头平替"叙事。要从客户基础四维度（户数 / AUM / 增速 / MAU）反向校核能级差，全面落后 2-3x = 不同档。

- 2026-05-08 平安银行 | "招行平替" 叙事不成立 | 案例：平银 vs 招行 全面对位——零售 AUM 3.87x / 私行客户 1.77x / 私行 AUM 2.47x / 财富管理手续费 2.76x / MAU 4.5x（招行 1.7 亿 vs 平银 3,820 万）—— 每个维度落后 1.5-3.9x。**全维度全面落后 = 不同档**，"招行平替"是错误叙事。**正确定位**：平银是"改革主动型股份行"，应与兴业（PB 0.45）/ 中信（PB 0.62）平视，而非招行（PB 0.84）平替（partial）
- **通用规则**：
  - "X 平替" 叙事必须做四维校核：客户/产品基础规模、获客/产品增速、活跃度/质量指标、单位经济（户均 / 单店）
  - 单维度落后 1.5x 内 = 真平替；2-3 维度落后 2x = 跟随者；全维度落后 2-3x+ = 不同档（不能平替）
  - 不同档时，估值差距应大致匹配能级差（如平银 PB 0.47 / 招行 0.84 = 0.56x ≈ 综合能级差倒数）
  - 适用：金融（招行 vs 平银 / 兴业 / 中信）、互联网（腾讯 vs 字节 / 网易）、消费（茅台 vs 五粮液 / 国窖）

## 待积累的盲点 / 待补维度

> 未来 deep-dive 时主动检查这些维度，积累案例后形成正式条目。

- **B2B vs B2C 业务区别**：仪器（华大智造）、半导体设备（北方华创/中微）、CXO（药明康德）等 to B 业务，客户名单 / 合同周期 / 单一大客户依赖度比销量数据更重要
- **渠道库存压力**：经销商提车 vs 终端零售的差距，反映渠道压力
- **复购 / NPS / 用户留存**：互联网、SaaS、医药 CXO、消费品都需要这个维度
- **创始人/管理层股权变化**：减持节奏、激励机制变化反映对未来的真实判断
- **关键供应商 / 关键客户的同期数据**：上游原材料供应商业绩 / 下游核心客户业绩，构成产业链交叉验证
