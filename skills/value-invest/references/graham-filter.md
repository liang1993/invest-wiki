# 格雷厄姆参考尺

> 本文件被 `SKILL.md` 的 Step 4C 引用。纯参考，**不影响估值结论**。
> 触发时机：Step 4B 之后、Step 5 之前。跳过条件：亏损股 / BVPS 为负 / 金融股（BVPS 口径不可比）。

## 核心公式

```
Graham Number = √(22.5 × EPS × BVPS)
```

- EPS 用 Step 1B.6 选定的 EPS（优先扣非）
- BVPS 用最新季报每股净资产
- 22.5 = 15（PE 上限）× 1.5（PB 上限）

**判定档位**：

| 当前价 / Graham | 判定 |
|---|---|
| < 0.67 | 深度低估 |
| 0.67 - 1.0 | 低估 |
| 1.0 - 1.5 | 合理偏贵 |
| 1.5 - 2.0 | 偏贵 |
| > 2.0 | 远超格雷厄姆门槛 |

## 防守型 7 条清单（打勾即可，不打分）

| # | 条件（已按 A股/港股放宽） |
|---|---|
| 1 | 营收 ≥ 50 亿 RMB / 20 亿港币 |
| 2 | 流动比率 ≥ 2（金融股跳过此项） |
| 3 | 近 10 年每年盈利 |
| 4 | 近 5 年每年分红（原 20 年放宽为 5 年） |
| 5 | 近 10 年 EPS 增长 ≥ 33% |
| 6 | 当期 PE ≤ 20（原 15 放宽） |
| 7 | PB ≤ 1.5 或 PE × PB ≤ 30（原 22.5 放宽） |

输出：`格雷厄姆 7 条清单覆盖：n/7`

## A股/港股已做的调整

- 股息记录：20 年 → 5 年
- PE 上限：15 → 20
- PE × PB：22.5 → 30
- 周期股 EPS 用近 5-7 年均值（避免周期顶/底失真）
- Graham Number 本身**不调整系数**（仅用比率看溢价，调整意义不大）
