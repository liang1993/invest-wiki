# Step 1B: 利润表分解与EPS基线

> 详细子流程，由 SKILL.md 的 Step 1B 概要章节按需加载。

**目的**：AKShare 的 `stock_financial_abstract` 只提供汇总指标（毛利率、净利率），无法看到利润表的每一行。对于存在大额非经常性损益、持续/终止经营分拆、SBC费用等复杂情况的公司，仅看汇总指标会导致EPS基线失真。本步骤通过读取年报利润表逐行拆解，建立真实的经营利润基线。

**适用条件**（满足任一即执行）：
- 归母净利润 与 扣非净利润 偏差 > 15%
- 存在终止经营/剥离业务
- 投资收益 / 公允价值变动 占净利润比例 > 10%
- AKShare 的毛利率与 (1 - 营业成本/营业总收入) 计算值不一致
- 公司有大额SBC（科技/CXO/创新药常见）

**不适用**：归母与扣非接近（偏差<10%）、利润结构简单的成熟企业（如茅台、海天）。

## 1B.1 获取年报利润表

通过 WebSearch 搜索"公司名 + 年报 + 利润表"，从巨潮/交易所/公司官网获取PDF年报，读取合并利润表页面。提取以下逐行数据（单位：亿元）：

| 税金及附加 | | |
| 销售费用 | | |

同时从年报中提取**非经常性损益明细表**（通常在"主要会计数据和财务指标"章节）。

## 1B.2 利润瀑布图

将利润表组织为层次化瀑布，标注每层占收入的百分比和同比变化：

```
营业收入              xxx亿   100%    +xx%
- COGS               (xxx)   -xx%    +xx%
= 毛利                xxx亿   xx.x%   +xx%    ← 真实毛利率（不依赖AKShare）
- 期间费用+税金        (xxx)   -xx%    ← 逐项列示
= 经营毛利             xxx亿   xx.x%   ← 核心经营能力
+ 其他/投资/FV等       xxx            ← 区分经常性 vs 一次性
= 营业利润             xxx亿
- 税/少数股东          (xxx)
= 归母净利润           xxx亿
```

## 1B.3 非经常性损益桥接

逐项分解从归母到扣非的路径：

```
归母净利润           xxx亿
减:非经常性损益      -xxx亿
  ├ {项目1}          +/-xxx   ← 逐项列出，标注是否来自年报非经常明细表
  ├ {项目2}          +/-xxx
  ├ ...
  ├ 所得税影响        -xxx
  └ 少数股东影响      -xxx
= 扣非归母净利润     xxx亿    ← 用于PE估值的EPS基准

加:SBC add-back     +xxx亿   ← 从"扣除股份支付影响后的净利润"反推
加:其他调整          +xxx亿
= 经调整Non-IFRS     xxx亿    ← 公司自定义口径（如有）
```

## 1B.4 核心经营利润验证

反推验证扣非净利润与利润表的一致性：

```
核心营业利润 = 营业利润 - 非经常性损益(税前合计)
核心利润总额 = 核心营业利润 ± 营业外
核心所得税 = 总所得税 - 非经常性税影响
核心净利润 = 核心利润总额 - 核心所得税
核心归母 = 核心净利润 - (少数股东 - 非经常少数股东影响)

验证:核心归母 ≈ 扣非归母（偏差应<1%）
→ 输出核心有效税率 = 核心所得税 / 核心利润总额
```

如果验证不通过（偏差>2%），检查是否有遗漏的非经常性项目。

## 1B.5 利润驱动因子排序

对比本期与上期，按对扣非利润的贡献绝对值排序，列出前5大驱动因子：

```
| 排名 | 驱动因子 | 对扣非利润的贡献(亿) | 说明 |
|------|--------|-------------------|------|
| 1 | {如:毛利率提升} | +xx | {具体数据} |
| 2 | {如:收入增长} | +xx | |
| ... | | | |
```

这一步的输出直接决定了远期分析中哪些变量最需要重点预测。

## 1B.6 EPS基线决策

根据分解结果，决定PE估值使用哪个EPS：

| 条件 | 使用的EPS | 理由 |
|------|---------|------|
| 归母与扣非偏差<10% | 归母EPS | 差异不大，用报表口径 |
| 偏差10-30%，非经常主要为正向 | 扣非EPS | 非经常收益不可持续 |
| 偏差10-30%，非经常主要为负向 | 归母EPS | 非经常损失已过去 |
| 偏差>30% | 扣非EPS | 剔除大额一次性 |
| 存在持续/终止经营分拆 | 持续经营口径的扣非EPS | 终止业务已剥离 |

输出：选定的EPS值及理由，作为后续 Step 3（PE模型）和 Step 4B（远期分析）的输入。
