# Step 3: 合理PE定价模型（因子推导）

> 详细子流程，由 SKILL.md 的 Step 3 概要章节按需加载。

> ⚠️ **触发条件**：仅当 Step 0 判定主估值 = PE 时执行本节。
> - P/S 主估值的公司（弱周期成长，如汽车/家电）→ 跳到 Step 4 的 P/S 模型
> - PB 主估值的公司（真周期/金融）→ 跳到 Step 4 的 PB 模型
> - PEG 主估值的公司（高成长）→ 跳到 Step 4 的 PEG 模型

## 3.1 行业基础PE

根据行业属性确定基础PE：

| 行业类型 | 基础PE | 理由 |
|----------|--------|------|
| 必选消费（白酒、调味品、乳制品） | 20x | 需求稳定，现金流可预期 |
| 可选消费（家电、汽车、服装） | 15x | 周期性略强 |
| 医药（创新药、CXO） | 20x | 长周期成长 |
| 医药（仿制药、中药） | 15x | 成长性有限 |
| 科技（互联网、软件） | 20x | 高成长但波动大 |
| 科技（硬件、半导体） | 18x | 周期+成长 |
| 金融（银行） | 8x | 高杠杆、低增速 |
| 金融（保险、券商） | 12x | 周期性强 |
| 周期（钢铁、煤炭、化工） | 8x | 强周期，利润波动大 |
| 公用事业（电力、水务） | 12x | 稳定但低增速 |
| 地产 | 6x | 行业下行周期 |

如果行业不在列表中，根据行业特征类比选取。

## 3.2 加分因子

| 因子 | 条件 | 加分 |
|------|------|------|
| 护城河-品牌 | 毛利率>70%且5年波动<3% | +5x |
| 护城河-品牌 | 毛利率>50%且5年波动<5% | +3x |
| 护城河-规模 | 营收为行业第二名2倍以上 | +3x |
| 护城河-规模 | 营收为行业第一 | +2x |
| ROE优秀 | ROE>25%且近3年逐年提升 | +3x |
| ROE优秀 | ROE>20%且近3年稳定 | +2x |
| ROE良好 | ROE>15%且近3年稳定 | +1x |
| 现金流优秀 | 经营现金流/净利润>1（近3年均值） | +2x |
| 现金流良好 | 经营现金流/净利润>0.8（近3年均值） | +1x |
| 高成长 | 近3年净利CAGR>20% | +3x |
| 稳定成长 | 近3年净利CAGR 10%-20% | +2x |
| 分红优秀 | 股息率>3%且连续5年以上 | +2x |
| 分红良好 | 股息率>2%且连续3年以上 | +1x |

## 3.3 减分因子（财务风险）

| 因子 | 条件 | 减分 |
|------|------|------|
| 增速下滑 | 最新增速<近3年均值的50% | -3x |
| 增速下滑 | 最新增速<近3年均值的70% | -2x |
| 负增长 | 最新营收或净利负增长 | -3x |
| 现金流恶化 | 经营现金流/净利润<0.5 | -2x |
| 高负债 | 资产负债率>60%（非金融） | -2x |
| 应收异常 | 应收增速>营收增速×1.5 | -2x |
| 利润率下滑 | 净利率连续2年下降 | -1x |
| ROE下滑 | ROE连续2年下降 | -1x |

## 3.4 减分因子（行业/商业模式风险）

根据行业匹配预设风险因子，再结合具体公司微调：

**白酒行业预设风险：**
| 风险 | 减分 | 说明 |
|------|------|------|
| 消费人群萎缩 | -2x | 人口负增长+年轻人饮酒减少，不可逆 |
| 政策风险 | -2x | 消费税改革、限制三公消费 |
| 消费场景依赖 | -1x | 政商宴请占比高 |
| 替代品竞争 | -1x | 低度酒、洋酒侵蚀 |
| 品类集中 | -1x | 核心大单品收入占比>60%时触发 |
| 渠道库存风险 | -1x | 经销商模式，库存周期放大波动 |

**医药行业预设风险：**
| 风险 | 减分 | 说明 |
|------|------|------|
| 集采政策 | -3x | 仿制药集采持续推进 |
| 医保控费 | -2x | 医保谈判持续压价 |
| 研发失败 | -2x | 创新药管线不确定性 |
| 审批风险 | -1x | 政策审批周期长 |
| 技术迭代 | -1x | 新疗法可能颠覆现有产品 |

**科技行业预设风险：**
| 风险 | 减分 | 说明 |
|------|------|------|
| 技术迭代快 | -2x | 护城河可能被颠覆 |
| 监管风险 | -2x | 反垄断、数据安全 |
| 客户集中 | -1x | 大客户依赖度>30%时触发 |
| 人才流失 | -1x | 核心技术人员依赖 |

**地产行业预设风险：**
| 风险 | 减分 | 说明 |
|------|------|------|
| 行业下行周期 | -3x | 需求长期收缩 |
| 政策压制 | -2x | 房住不炒 |
| 高杠杆 | -2x | 行业普遍高负债 |
| 流动性风险 | -2x | 债务到期压力 |

**周期行业预设风险：**
| 风险 | 减分 | 说明 |
|------|------|------|
| 强周期波动 | -2x | 利润波动幅度大 |
| 产能过剩 | -2x | 供给端竞争激烈 |
| 政策调控 | -1x | 环保、产能限制 |

**金融行业预设风险：**
| 风险 | 减分 | 说明 |
|------|------|------|
| 资产质量 | -2x | 坏账/信用风险不透明 |
| 利率风险 | -1x | 净息差收窄 |
| 监管变化 | -1x | 资本充足率等要求趋严 |

**其他行业：** 根据行业特征动态判断，给出2-5项核心风险，每项-1x到-3x。

## 3.5 公司特有风险微调

在行业预设基础上，根据具体公司情况微调：
- 如果公司在某项风险上明显优于行业（如茅台受替代品竞争影响最小），可减轻该项减分
- 如果公司有行业预设之外的特殊风险（如管理层变动、诉讼等），额外减分
- 微调时给出理由

## 3.6 合理PE计算

```
合理PE = 行业基础PE + 加分合计 + 减分合计
```

**下限保护**：合理PE不低于8x（除非公司基本面严重恶化）
**上限保护**：合理PE不超过50x（防止过于乐观）
