# Step 4: 多模型估值与交叉验证

> 详细子流程，由 SKILL.md 的 Step 4 概要章节按需加载。

不同企业特征适用不同估值模型。先判断企业类型，再选择对应的模型组合。

> ⚠️ **重要：本节是 Step 0 判完类后的"查表确认"，不是独立判类**。
> Step 0 已经基于 [估值方法选择框架](../../../wiki/strategies/估值方法选择框架.md) 5 问决策树定下了主估值方法（PE / PB / P/S / PEG），本节的 7 类企业表只是用于：
> 1. 确认 Step 0 选定的主估值与本节一致
> 2. 在 PE 主路径下，进一步细分到"利润改善型/高分红型/困境反转型"以微调因子
>
> 不要在本节重新判类——若与 Step 0 矛盾，回到 Step 0 重判，不在 Step 4 自行改变主估值。

## 4.1 企业分类与模型选择

| 企业类型 | 特征 | 主模型 | 辅助模型 | DCF权重 |
|----------|------|--------|---------|---------|
| **现金牛型** | 现金流稳定、增速平稳、商业模式成熟（如茅台、长江电力） | PE定价 | DCF（高可靠性） | 与PE等权，取均值 |
| **利润改善型** | 现金流波动但利润率持续提升（如青啤、海天味业转型期） | PE定价 | DCF（中可靠性） | 仅作参考，不参与定价，但若DCF远低于PE模型需标注风险 |
| **高成长型** | 营收高增速（>25%）、利润可能尚未释放（如早期新能源） | PEG定价 | PE定价 | 不适用（现金流不稳定） |
| **真周期型** | 营收+销量同步周期波动（钢铁/煤炭/水泥/有色等大宗品） | PB定价 | PE仅在周期底部参考 | 不适用（FCF波动太大） |
| **弱周期成长型** ★ | **营收单边上行 + 利润率周期波动**（汽车/家电/部分制造）| **P/S定价** | 中周期PE（营收×正常化净利率×PE）+ SOTP | 不适用（FCF波动）|
| **高分红型** | 低增速但分红稳定（如银行、高速公路） | DDM定价 | PE定价 | DCF作为补充 |
| **困境反转型** | 当前亏损或微利，但有明确反转预期 | 目标利润×合理PE | — | 不适用 |

> ⚠️ **"真周期型" vs "弱周期成长型" 的关键区别**：
> - 真周期：营收+销量同步周期波动，萎缩年常见 → PB 主
> - 弱周期成长：营收单边上行（无萎缩年），仅利润率受行业内卷扰动 → **P/S 主**（汽车/家电请用此类）
> - 长城汽车 2026-04-28 案例：营收 8 年 +125% 单边上行（仅 2019 -3%），错归"周期型"用 PE/PB 会双重打折，应归"弱周期成长型"用 P/S

## 4.2 各模型计算方法

### PE模型（所有类型通用）

```
合理价值_PE = 合理PE × TTM EPS
```

合理PE由 Step 3 的因子模型推导。

### DCF模型（现金牛型/高分红型适用）

两阶段自由现金流折现：

```python
# 参数选择原则（保守为主）
# 自由现金流 = 经营现金流 - 资本支出
# 5年增长率：取近3年FCF增速和净利增速的较低者，再打8折
# 永续增长率：消费品2-3%，周期品1-2%，公用事业1-2%
# 折现率：
#   高确定性企业 8-9%（茅台、长电等）
#   中确定性企业 9-10%
#   低确定性企业 10-12%

def dcf(fcf, growth_5y, terminal_growth, discount_rate, shares):
    # 阶段1：5年预测期
    pv_sum = sum(fcf * (1+growth_5y)**i / (1+discount_rate)**i for i in range(1,6))
    # 阶段2：永续价值
    terminal_fcf = fcf * (1+growth_5y)**5 * (1+terminal_growth)
    terminal_pv = terminal_fcf / (discount_rate - terminal_growth) / (1+discount_rate)**5
    return (pv_sum + terminal_pv) / shares
```

**附敏感性分析**：折现率±1%、永续增长率±1%的3×3矩阵——DCF 对这两个参数极敏感（永续增长率从 2% 改到 3% 估值差 25%+），不附矩阵就看不出"合理价 X 元"的可信区间。

### DDM模型（高分红型适用）

```
合理价值_DDM = 下一年预期股息 / (要求回报率 - 股息增长率)

# 要求回报率：8-10%
# 股息增长率：取近5年股息CAGR，上限不超过净利增速
```

### PEG模型（高成长型适用）

```
合理PE_PEG = 净利增速(%) × PEG倍数
# 保守PEG = 0.8，合理PEG = 1.0
合理价值_PEG = 合理PE_PEG × TTM EPS
```

### PB模型（真周期型 / 金融型适用）

```
# 用历史PB区间定价
# 买入价 = 历史低点PB中位数 × 每股净资产
# 卖出价 = 历史高点PB中位数 × 每股净资产
```

### P/S模型（弱周期成长型适用）★

> 详见 [估值方法选择框架](../../../wiki/strategies/估值方法选择框架.md) 的 P/S 推导通用模板和子行业基准。

```
合理 P/S = 全球可比龙头基准 P/S
        ± 规模因子（小公司 -0.10~-0.20 / 巨头 +0.10）
        ± 成长因子（出海+高端化等结构变化 +0.05~+0.15）
        ± 净利率因子（短期承压 -0.10 / 长期高于同行 +0.10）
        ± 治理 / 公司特有因子 ±0.05

合理价值_PS = 合理 P/S × 中周期营收（前瞻 1-2 年）/ 总股本
```

**P/S 法的三个支撑**（缺一会导致估值飘）：

1. **同行可比表**：列出全球 + 国内 5-8 家可比公司的 P/S（市值/营收），并给出本公司在分布中的位置——单只标的的合理 P/S 不能拍脑袋，得有同行参照系。优先使用 [估值方法选择框架](../../../wiki/strategies/估值方法选择框架.md) 的子行业 P/S 基准；wiki 没列的子行业自行实证补充
2. **历史 P/S 中枢检查**：5-10 年区间散点，标注峰谷年份并剔除主题泡沫年份（如汽车股 2020-2021 PE 60x 等极端值）——不剔除会让中枢被泡沫拉高
3. **三模型加权**：P/S 50% + 中周期 PE 30% + SOTP 20%（详见 4.3）——三模型互相校验比单模型稳健

### 中周期 PE 模型（弱周期成长型辅助）

```
# 区别于 Step 3 的合理 PE 因子推导：
# 中周期 PE 用"营收 × 正常化净利率 × 合理 PE / 总股本"
# 避免 TTM PE 在利润率底位时的双重打折

合理价值_中周期PE = 中周期营收 × 正常化净利率 × 合理 PE / 总股本

# 正常化净利率：取历史 5-10 年中位数，避开峰谷年份
# 合理 PE：弱周期成长股给 14-15x（vs 真周期股 8-10x）
```

### SOTP 分部估值（弱周期成长型 + 多业务多市场公司辅助）

```
# 按业务/市场拆分，每块给不同 P/S 或 PE
# 示例（长城汽车）：
# 国内主流 营收 × P/S 0.5x（同长安）
# 国内高端 营收 × P/S 1.5x（同理想）
# 海外     营收 × P/S 1.2x（出海溢价）
# 合计 → 总市值 / 总股本 → 合理价
```

## 4.3 交叉验证规则

根据企业类型，对多模型结果交叉验证：

**现金牛型**（PE + DCF 等权）：
```
综合合理价值 = (合理价值_PE + 合理价值_DCF) / 2
```
如果两个模型偏差>30%，需要检查假设是否合理并在报告中说明分歧原因。

**利润改善型**（PE为主，DCF参考）：
```
综合合理价值 = 合理价值_PE
# 但如果 DCF估值 < 当前股价，标注：
# "DCF估值(XX元)低于当前价，说明当前价格隐含了对未来利润率持续改善的预期，若改善不及预期存在下行风险"
```

**高成长型**（PEG为主，PE校验）：
```
综合合理价值 = 合理价值_PEG
# 若 合理价值_PE < 合理价值_PEG × 0.7，说明PE模型认为高成长不可持续，标注风险
# 注意：使用PEG为主模型时，增速已决定了合理PE，Step 3中的增长类加分因子（高成长+3x、稳定成长+2x）
# 在静态PE中应剔除，避免增速被双重计算。仅保留与增速无关的因子（护城河、现金流、风险等）。
```

**真周期型**（PB为主）：
```
综合合理价值 = 基于PB区间
# PE仅在行业周期底部（利润极低时）作为辅助参考
```

**弱周期成长型**（P/S 主 + 中周期 PE 辅 + SOTP 辅 三模型加权）★：

```
综合合理价值 = 合理价值_PS × 50%
            + 合理价值_中周期PE × 30%
            + 合理价值_SOTP × 20%
```

> ⚠️ **权重 + 阈值依据声明（双重局限）**：
> - **权重 50/30/20** 基于长城汽车 v3 单案例校准（与市场实际定价偏离 -16%）
> - **复查阈值 30% / 50%** 同样基于单案例拍脑袋
> - **两者都需要 3-5 个标的回测后再调**
>
> 在精度未稳定前，**真正起兜底作用的是"差异是否引发复查"这一行为**，而不是阈值本身的精度——若 P/S 与 PE 差距让你"感到不舒服"，无论是 25% 还是 35%，都应该复查。不必拘泥精确比例。

**复查规则（比权重精度更重要）**：

| 触发条件 | 行动 |
|---------|------|
| **P/S 与中周期 PE 差距 > 30%**（阈值待回测）| **优先信 P/S**（营收锚比净利率锚稳定）|
| **P/S 与中周期 PE 差距 > 50%**（阈值待回测）| **强制复查**：(a) 同行可比表是否选错对标？(b) 净利率假设是否过激进/保守？(c) SOTP 各分部 P/S 是否合理？|
| **三模型最终结果 vs 市场实际定价偏离 > ±20%** | 触发反向健康检查（Step 0 末尾），重新走决策树 |

**校准记录**（每新增一个弱周期成长股案例后更新）：
- 长城汽车 v3：P/S 25 / PE 22 / SOTP 26 → 加权 24 元 → vs 市场 20.23 = **-16% 偏离**（合格）
- 待补：第 2-5 个案例（美的/立讯/比亚迪等）回测后调整权重

**高分红型**（DDM + PE 等权）：
```
综合合理价值 = (合理价值_DDM + 合理价值_PE) / 2
# 注意：使用DDM为主模型时，分红已决定了定价基础，Step 3中的分红类加分因子（分红优秀+2x、分红良好+1x）
# 在静态PE中应剔除，避免分红贡献被双重计算。仅保留与分红无关的因子（护城河、成长性、风险等）。
```
