# Step 4B: 远期分析（1-2年前瞻估值）

> 详细子流程，由 SKILL.md 的 Step 4B 概要章节按需加载。

静态PE模型用的是TTM/最近年报EPS，对于业务结构正在变化的公司（如AI转型、新业务放量、补贴退出）会严重失真。远期分析通过分业务建模，估算1-2年后的利润水平和合理PE，折现回当前与静态模型交叉验证。

**适用场景**（任一成立时执行远期分析）：
- 公司正在经历业务结构显著变化（新业务收入占比快速提升或下降）
- 存在已知的临时性费用/收入（闪购补贴、一次性减值、并购整合费用）
- 核心业务增速与新业务增速方向相反（如电商放缓但云加速）
- 当前PE因临时因素显著偏离合理PE

**不适用**：
- 业务结构稳定、增速平稳的成熟企业（如茅台、长江电力），静态PE足够
- **真周期型企业**（如煤炭、钢铁、猪肉、大宗化工）：利润随行业周期大幅波动，1-2年的线性外推会严重失真。周期股应使用PB定价+周期位置判断，不做远期利润预测

**特别适用**：
- **弱周期成长型企业**（汽车/家电/部分制造）：营收单边上行但利润率周期波动，远期分析必须做——但**重点是营收建模而非利润，配合 Step 4 的 P/S 主估值**

**前置判断流程**：进入远期分析前，先确认企业类型（Step 0）。如果是**真周期型**，跳过整个Step 4B，直接进入Step 5。

## 4B.1 分业务营收建模

将公司营收拆解为主要业务板块（3-5个），逐个估算未来2个财年的增速：

```
| 业务板块 | 当前营收 | 占比 | FY+1增速 | FY+2增速 | 依据 |
|----------|---------|------|---------|---------|------|
| 板块A | xxx亿 | xx% | x% | x% | {具体理由} |
| 板块B | xxx亿 | xx% | x% | x% | {具体理由} |
| ... | | | | | |
| 合计 | xxx亿 | 100% | x% | x% | |
```

**增速估算原则（保守为主）：**
- 成熟业务：取近2年增速的较低值，除非有明确拐点信号
- 高增长业务：取管理层指引或分析师一致预期的80%（打折）
- 下滑业务：按最近2个季度的同比趋势线性外推
- 新业务（<2年历史）：不给高增速假设，按行业平均增速

## 4B.2 利润率预测

对每个业务板块预测目标利润率：

```
| 业务板块 | 当前OPM | FY+1 OPM | FY+2 OPM | 依据 |
|----------|--------|---------|---------|------|
| 板块A | xx% | xx% | xx% | {对标/趋势/天花板分析} |
| 板块B | xx% | xx% | xx% | |
| 集团费用 | -xx亿 | -xx亿 | -xx亿 | {总部/股权激励等} |
```

**利润率估算原则：**
- 改善中的业务：参照同行业成熟公司的利润率作为天花板，不超过天花板的90%
- 稳定业务：维持当前水平
- 恶化中的业务：按最近趋势外推，除非有明确的成本削减计划
- 临时性费用（如闪购补贴）：基准情景假设退出，悲观情景假设持续

## 4B.3 远期EPS计算

```python
# 远期经营利润 = Σ(各板块营收 × 各板块OPM) - 集团费用
# 远期净利润 = 远期经营利润 × (1 - 有效税率) + 预计投资收益（保守取0或近3年均值×50%）
# 远期EPS = 远期净利润 / 预计股本数（考虑回购）
```

**注意**：
- 投资收益/损失波动大的公司（如阿里、腾讯），远期净利润估算中投资收益取保守值（0或历史均值的50%）
- 有回购计划的公司，按近2年回购速度估算未来股本减少
- 有大额股权激励(SBC)的公司（科技公司常见），需估算每年SBC导致的股本净增加：
  净股本变化 = 回购减少的股本 - SBC新增的股本
  远期股本 = 当前股本 × (1 + 年净变化率)^N
  如果SBC占营收比例>5%，应在经营利润中扣除SBC费用（如果财报未扣除）

## 4B.4 因子调整SOTP PE

远期合理PE不能简单用行业通用PE加权——那会丢失公司特有的质量因子（护城河、现金流、风险）。正确做法是先算原始SOTP，再逐项审查静态因子是否仍然适用。

### 第一步：增速调整SOTP PE

每个板块的基础PE应根据该板块的预期增速进行调整。高增速板块理应获得更高PE，低增速板块则打折。

```
各板块远期PE = 行业基础PE × 增速调整系数

增速调整系数（线性插值公式）：
- <3%: 系数 = 0.8
- 3%-25%: 系数 = 0.8 + (growth - 0.03) / 0.22 × 0.7  （从0.8线性增长到1.5）
- >25%: 系数 = 1.5（封顶）

参考区间表（实际计算用上述线性插值，此表仅供快速参考）：
| 板块预期增速(FY+1~FY+2均值) | 参考系数 | 理由 |
|---------------------------|---------|------|
| <3% (近乎零增长) | ×0.8 | 价值消耗 |
| 3-8% (低增长) | ×0.8~1.0 | 行业平均 |
| 8-15% (稳健增长) | ×1.0~1.2 | 高于行业 |
| 15-25% (高增长) | ×1.2~1.5 | 增长溢价 |
| >25% (超高增长) | ×1.5 | 高成长溢价，但PE上限50x |

增速调整SOTP PE = Σ(各板块FY+2营收占比 × 各板块增速调整后PE)
```

**原则**：调整系数反映的是增速对估值倍数的影响，不是对利润的影响（利润已在EPS中）。一个增速18%的广告业务和增速3%的本地生活，即使利润率相同，前者也应该获得更高PE。

### 第二步：因子存续性审查

逐项检查 Step 3 中的每个加分/减分因子，判断其在远期分业务模型中是否已被"吸收"：

| 吸收状态 | 含义 | 处理方式 |
|----------|------|---------|
| **已吸收** | 因子效果已体现在板块增速或利润率预测中 | 不重复计入 |
| **部分吸收** | 因子效果部分体现在板块指标中，但有超越板块指标的公司层面溢价 | 按50%折扣计入 |
| **未吸收** | 因子完全是公司层面的系统性优势/风险，无法通过板块指标体现 | 全额计入 |

**常见因子的典型判断：**

| 因子类型 | 典型吸收状态 | 理由 |
|----------|------------|------|
| ROE优秀 | 已吸收 | ROE高是因为利润率高+资产效率高，这些已经在板块OPM和增速中体现 |
| 高成长/稳定成长 | 已吸收 | 增速已经是板块预测的输入变量 |
| 毛利率护城河 | 已吸收 | 毛利率水平已在板块OPM中体现 |
| 生态协同/网络效应 | 部分吸收 | 对各板块增速有贡献（已吸收），但跨板块协同、用户锁定效应超越单一板块指标（未吸收部分） |
| 规模优势 | 部分吸收 | 市场份额体现在营收（已吸收），但规模带来的议价能力和壁垒超越营收数字（未吸收部分） |
| 现金流优秀 | 部分吸收 | 利润率高意味着现金流基础好（已吸收），但OCF/NI>1的运营资本管理能力是公司层面的（未吸收部分） |
| 分红优秀 | 未吸收 | 分红政策是公司决策，不在板块指标中 |
| 监管风险 | 部分吸收 | 对增速的压制已在板块增速中考虑（已吸收），但黑天鹅式政策冲击无法用增速预测覆盖（未吸收部分） |
| 地缘政治风险 | 已吸收 | 板块基础PE使用的是中国公司估值水平，已内嵌地缘折价。若局势进一步恶化，应调整板块基础PE而非叠加额外因子 |
| 行业竞争加剧 | 已吸收 | 竞争影响已反映在板块增速和利润率预测中 |

### 第三步：计算因子调整后SOTP PE

输出格式：

```
| 静态因子 | 静态PE贡献 | 吸收状态 | 远期PE贡献 | 理由 |
|----------|-----------|---------|-----------|------|
| 网络效应 | +3x | 部分吸收(50%) | +1.5x | 跨板块协同超越单一板块 |
| 规模优势 | +3x | 已吸收 | 0x | 市场份额已在营收中 |
| ROE稳定 | +2x | 已吸收 | 0x | 已在利润率中 |
| 监管风险 | -2x | 部分吸收(50%) | -1x | 黑天鹅部分未在增速中 |
| 地缘政治 | -2x | 未吸收 | -2x | 系统性外部风险 |
| ... | | | | |
| 合计调整 | | | +Nx / -Nx | |

因子调整后SOTP PE = 原始SOTP PE + 未吸收因子合计
```

**与静态PE的关系**：
- 如果因子调整后SOTP PE与静态PE差异<3x → 两个模型一致性好，可互相验证
- 如果差异>5x → 需要检查：是因子模型"质量溢价堆叠"过度，还是SOTP遗漏了重要因子
- 一般来说，因子调整后SOTP PE应低于静态因子PE（因为部分因子被吸收），但不应低太多（>8x差距说明有问题）

## 4B.5 远期目标价与隐含回报率

```
远期目标价 = 远期合理PE × 远期EPS
当前价隐含年化回报率 = (远期目标价/当前价)^(1/N) - 1  # N=1或2年

# 加上预期分红：
含息年化回报率 = 隐含回报率 + 当前股息率
```

**回报率判断标准：**

| 含息年化回报率 | 判断 | 对买入决策的影响 |
|---------------|------|----------------|
| >20% | 显著低估 | 可直接买入，即使静态模型未到安全边际买入价 |
| 15-20% | 有吸引力 | 值得建仓，但控制仓位 |
| 10-15% | 合理回报 | 持有，但不追加 |
| 5-10% | 回报偏低 | 谨慎，寻找更好标的 |
| <5% | 无吸引力 | 不建议买入，已持有考虑减仓 |

## 4B.6 与静态模型交叉验证

```
| 估值方法 | 合理价值/目标价 | 权重 | 说明 |
|----------|---------------|------|------|
| 静态PE模型 | xx元 | 参考 | Step 4 结果 |
| 远期估值(FY+1) | xx元 | 主要 | 远期PE × 远期EPS，折现1年 |
| 远期估值(FY+2) | xx元 | 辅助 | 远期PE × 远期EPS，折现2年 |
| DCF模型 | xx元 | 参考 | Step 4 结果 |
```

**验证规则**：
- 如果远期估值显著高于静态估值(>20%)：说明当前利润是低点，公司在"蓄力"期，静态PE低估了公司 → 综合估值上修
- 如果远期估值显著低于静态估值(>20%)：说明当前利润是高点或不可持续 → 综合估值下修，提示风险
- 如果两者接近(±20%)：业务结构稳定，静态模型可靠

**综合估值（加入远期分析后）**：
```
# 有远期分析时：
综合合理价值 = 静态合理价值 × 40% + 远期FY+1折现价值 × 40% + 远期FY+2折现价值 × 20%

# 折现率 = Step 4 DCF中使用的折现率
```

## 4B.7b 底部P&L重建交叉验证

**前置条件**：Step 1B 已完成利润表分解。未执行则跳过。

**方法**：用 Step 1B 的利润瀑布图作为基线，对每个P&L行项独立估算 FY+1（表格结构同 Step 1B.2，增加"FY+1E"和"假设依据"列）。核心税率沿用 Step 1B.4 的计算值。

**逐行估算要点：**
- **毛利率**：用持续经营口径（可能高于总口径）
- **管理费用**：单独估算内含的 SBC 变化
- **财务费用**：分拆利息收支和汇兑损益，不能简单外推（如药明康德2024→2025从-7.87→+3.98亿）
- **减值损失**：仅预测经常性部分（应收坏账、存货），剔除非经常性（商誉、长期资产）

**交叉验证**：底部重建结果与 top-down（净利率×收入）、增速外推（扣非×(1+g)）三法对比。偏差<5%取均值；5-15%以底部重建为主并解释原因；>15%逐项排查假设。

**输出**：最终 FY+1/FY+2 扣非EPS，替换 4B.3 初始估计。
