# 页面模板

创建新 wiki 页面时，按以下模板的格式撰写。

## 个股模板

```markdown
# {股票名称} ({代码})

## 基本信息
- 市场：{A股/美股/港股}
- 行业：{所属行业}
- 市值：{最近已知市值}

## 投资逻辑
{核心看多/看空理由，简洁列出}

## 财务亮点
{关键财务指标：营收、利润、ROE、PE 等趋势}

## 风险点
{主要风险因素}

## 相关链接
- 行业：[{行业名}](../sectors/{行业名}.md)
- 相关宏观：[{主题}](../macro/{主题}.md)

## 信息来源
- {YYYY-MM-DD} {来源描述} — raw/{路径}

## 更新记录
- {YYYY-MM-DD} {更新内容摘要}
```

## 基金/ETF 模板

```markdown
# {基金名称} ({代码})

## 基本信息
- 类型：{指数/主动/混合/债券}
- 跟踪指数：{如适用}
- 费率：{管理费/托管费}
- 规模：{最近已知规模}

## 配置理由
{为什么选择这只基金}

## 持仓特点
{前十大持仓、行业分布等}

## 风险收益特征
{历史波动率、最大回撤、年化收益等}

## 相关链接
- 同类基金：[{名称}](../funds/{名称}.md)
- 相关行业：[{行业}](../sectors/{行业}.md)

## 信息来源
- {YYYY-MM-DD} {来源描述}

## 更新记录
- {YYYY-MM-DD} {更新内容摘要}
```

## 宏观经济模板

```markdown
# {主题名称}

## 概述
{主题简介和当前状态}

## 关键数据
{重要指标和时间线}

## 对投资的影响
{如何影响各类资产}

## 历史规律
{历史上类似情景的表现}

## 相关链接
- [相关主题](../macro/xxx.md)
- 受影响的行业：[{行业}](../sectors/{行业}.md)

## 信息来源
- {YYYY-MM-DD} {来源描述}

## 更新记录
- {YYYY-MM-DD} {更新内容摘要}
```

## 行业/板块模板

```markdown
# {行业名称}

## 行业概述
{行业特征、市场规模、增长逻辑}

## 核心公司
- [{公司名}](../stocks/{公司名}.md) — {一句话定位}

## 驱动因素
{行业增长/衰退的关键驱动力}

## 估值水平
{行业整体估值状态}

## 风险因素
{行业面临的主要风险}

## 相关链接
- 宏观背景：[{主题}](../macro/{主题}.md)

## 信息来源
- {YYYY-MM-DD} {来源描述}

## 更新记录
- {YYYY-MM-DD} {更新内容摘要}
```

## 投资日志模板

```markdown
# {YYYY-MM-DD} 投资日志

## 市场概览
{当日市场主要表现}

## 操作记录
- {买入/卖出} {标的} {数量} @ {价格} — {理由}

## 思考与复盘
{当日投资思考}

## 关注事项
{后续需要关注的事件或指标}
```
