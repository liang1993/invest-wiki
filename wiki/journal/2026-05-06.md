# 2026-05-06 SOX 25 日涨 50% 与 Hyperscaler Capex 拐点观察

## 三、判断——不是 1999，但已是"准备好的火药"

**与 2000 年互联网泡沫的关键差异**：

| 维度 | 2000 互联网 | 2026 半导体 |
|---|---|---|
| 盈利 | 大量公司无利润 | NVDA / AVGO 真金白银 + 正 FCF |
| 估值 | PS / PE 都在天上 | SOX 已 price perfection |
| 需求 | VC 烧钱 + 散户 | hyperscaler capex |
| 兑现度 | 营收虚 | GOOG Cloud +63%、backlog $462B 真兑现 |
| 风险点 | 收入虚 | **capex 周期性，边际放缓即触发 EPS 下修** |

**类比 [2026-04-27 系统性风险自检](2026-04-27.md) 框架**：当前是"估值已极端 + 流动性仍宽松"——**类 2017** 状态，不是 1999。1999 的特征是"估值极端 + YC 倒挂 + Fed 加息末期"，三者齐亮才是历史顶；当前后两者都不亮。

**但需警惕的"拐点信号"已开始出现**：

1. **Capex 加速度可能见顶**：Q1 同比 +72%、环比仅 +4%，环比已大幅收窄
2. **MSFT 自曝 $25B / $190B（13%）来自组件涨价**：被动加投占比明显，capex 总量"虚胖"
3. **市场对 capex 上调首次给出负反馈**：Meta 跌 6%，过去三年这种反应几乎没出现过
4. **三年 60%+ 增速 → 数学上不可持续**：2027 capex 要再加 $400B+，自由现金流将受压

## 四、Burry 期权时点的逻辑

Burry 选 **2027/1 到期**对得上：
- 2026 年报披露 → 2027 capex 指引出炉
- 若 2027 进入"高位平台"或下修 → 卖方共识 EPS 下修 → 60× PE × 下修 EPS = 双杀

这是"capex 见顶"假设最合理的兑现窗口。**他不是赌短期回调，是赌一年内 capex 周期见顶。**

## 五、传导路径——不动手，但加监控

美股半导体直接敞口非本组合主要风险源，主要风险通过 **A 股半导体板块（97% 估值分位，与 SOX 高 β）**间接传导。

**操作判断**：
- 不必现在清仓任何与 AI 相关的标的，capex 仍加速
- 但要把"半导体"加入 [系统性风险防御](../strategies/系统性风险防御.md) 的**结构性领先指标**——SOX 若率先回调 10%+，可能比 CAPE / HY OAS 早 3-6 个月给信号
- 关注 [腾讯](../stocks/internet/腾讯控股.md) / [小米](../stocks/internet/小米集团.md) / [中芯国际](../stocks/tech-hardware/中芯国际.md) 等持仓与 AI 主题的 β（高 β 名称在 SOX 拐点时跌幅会放大）
