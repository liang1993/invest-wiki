# 2026 美以伊冲突

## 概述

2026年2月28日，美国与以色列联合发动对伊朗的大规模海空军事打击，伊朗最高领袖**哈梅内伊在空袭中身亡**。冲突持续约40天，伊朗于3月4日关闭霍尔木兹海峡（全球25%海运石油 / 20% LNG必经之地）。4月8日在巴基斯坦斡旋下达成**两周临时停火**，但4月11-12日伊斯兰堡最高级别谈判（万斯 vs 卡利巴夫）破裂。4月18日伊朗部分开放领空，伊朗航空、马汉航空开始case-by-case复飞 → **战术性休战，非战略性和解**。

**当前关键观察窗口**：4-22 停火期到期 → 4-28~29 和谈再破裂 → 4-30 Brent 触及战时新高 $126 → **5-1 UAE 退出 OPEC+（独立增产）→ 5-5 霍尔木兹冲突未破停火 → 5-6 油价大跌**：Brent 收 $109.87（-4%）/ WTI 收 $102.27（-3.9%），盘中曾跌至 Brent $96.86 / WTI $88.82。美防长 Hegseth 5-6 称"近一个月前开始的停火仍在维持"，伊朗外长 Araghchi 称谈判"正在取得进展"——叙事从 4-30"军事升级 vs 谈判突破二选一" 转入 **"地缘溢价快速回吐 + 谈判低烈度推进"**。

## 谈判核心矛盾（4/11-12 破裂原因）

| 议题 | 美方立场 | 伊方立场 |
|------|---------|---------|
| 停火性质 | 临时停火 | 必须永久结束冲突 |
| 霍尔木兹 | 美海军封锁过境船只 | 革命卫队视任何美舰接近为违反停火 |
| 核问题 | 完全弃核+核查 | 保留民用核权利 |
| 美军中东驻留 | 维持驻军 | 要求美军撤出地区 |
| 制裁/冻结资产 | 部分有条件解冻 | 全面解冻+战争赔偿 |
| 黎巴嫩问题 | 真主党解除武装 | 拒绝 |

**霍尔木兹日均通过量**：从 2 月日均 2,000+ 万桶**降至 380 万桶**（-81%），IEA 称为"史上最大供应冲击"。

- **EIA上调2026全年均价预测至 $96/桶**（Q2峰值预期 $115）
- **悲观情景**：若霍尔木兹长期封锁，Brent中枢$120-150
- **乐观情景**：若达成持久协议，回归$70-85区间
- 中信、Allianz、Oxford Economics共识：**$80美元/桶是经济承受拐点，超出后全球通胀+衰退风险显著放大**

### 2. LNG / 天然气

- 卡塔尔LNG产能因Ras Laffan被袭**-17%**
- **亚洲LNG现货价+140%**
- 欧洲二次能源危机（恰逢储气量仅30%低位）
- QatarEnergy对所有LNG出口宣布**不可抗力**

### 3. 航运与保险（被忽视的关键传导链）

- **战险保费 > 船舶价值 5%**（伦敦劳埃德），商业过境基本不可行
- **超过200艘油轮+LNG船锚泊海峡外**
- 保险撤出（而非物理封锁）成为实际中断主因
- 全球油气运输船日租金大涨，VLCC日租从~$3万→$15万+

### 4. 黄金：避险逻辑被美元强势压制

| 日期 | Gold ($/oz) | 备注 |
|------|------------|------|
| 2026-02-28 | 5,296 | 冲突前 |
| 2026-03-01 | 5,423 | 初期避险冲高 |
| 2026-03-03 | 5,085 | 避险sell-off，-6% |
| 2026-04-15 | ~4,815 | 谈判预期+美元强势压制 |
| 2026-04-18（参考） | ~4,760 | 仪表盘记录 |

- **反直觉现象**：美元（而非黄金）成为首选避险资产，金价反受压制
- 高油价 → 通胀预期上行 → Fed降息推迟 → 实际利率上行 → 金价承压
- **长期目标价**：JPM预测年底 $6,300/oz，DB维持 $6,000/oz
- 央行连17个月购金（2026-04仍延续），结构性多头未变

### 5. 全球股市

- **冲突初期**：亚洲、欧洲、美国主要市场普跌；韩国、东京暴跌；A股短期情绪冲击+板块分化
- **A股受益板块**：油气开采/油服设备、贵金属/黄金、国防军工、煤炭/煤化工
- **A股承压板块**：航空、消费、地产链、对外依存度高的化工
- 多数券商共识：**中长期A股运行趋势不会因地缘事件根本改变**，主线仍回归国内基本面

### 6. 美元 / 美债 / 利率

- **美元指数走强**（避险首选+"石油美元"清算需求）
- 美国3月CPI跳升至 **3.3%**（前值2.4%），月率0.9%为近年最高，**汽油+18.9% MoM** 是主因
- 核心CPI仅2.6%，Fed可能"look through"能源冲击
- **Fed降息窗口进一步推迟**，2026年降息预期大幅收敛

### 7. 食品 / 民生（地区灾难）

- 海合会国家**80%卡路里**依赖霍尔木兹通道
- 3月中旬 **70%食品进口中断**
- 伊朗袭击海水淡化厂，**威胁科威特/卡塔尔99%饮水来源**
- 从"金融冲击"演变为"人道主义危机"

## 对中国的影响

### 直接经济影响
- 2025年中国石油消费 **7.5亿吨**，进口依存度 **72.7%**
- 油价上涨50%（从$70→$105），每年多支出 **>$1000亿美元**
- 战略储备充足，可短期平抑波动；多元化进口降低对单一货源依赖

### 战略机遇：石油人民币加速
- **2026年1月起，中伊石油贸易100%人民币结算**
- 伊朗央行持人民币头寸 **1800亿元**
- **CIPS单日交易突破1.22万亿元**（清算时间从数天缩至秒级）
- 人民币在动荡中显现 **避险属性**，跑赢主要非美货币
- 离岸人民币国际化获结构性推力

### 航运/能源安全
- 部分中国船只因双边关系仍获得通行
- 加速推进**中亚陆上能源通道**（中俄/中哈/中缅油气管道）
- 中欧班列分流部分集装箱货物

## 投资启示

### 短期（停火期内 / 2-4周维度）
- **风险资产 risk-on**：A股/港股情绪修复
- **油价**：中枢回落至$80-95，但下方有强支撑（地缘溢价不会归零）
- **黄金**：短期承压，回调即买入逻辑未变
- **不建议押注极端方向**："持续战火"和"完全解决"两端概率都不高

### 中期（4/22停火到期 / 1-3个月维度）
**核心判断**：4月22日前后是关键决策节点
- **续签停火（基准情形 50%）**：油价$80-90，避险逻辑缓和，A股有限回升
- **谈判进一步破裂、冲突复燃（30%）**：Brent冲击$120+，黄金重新走强，全球避险
- **达成持久协议（20%）**：油价回归$70-85，制裁部分解冻，伊朗原油回归市场（中长期利空油价）

### 长期影响（半年+）
- 中东能源安全问题**长期化**，地缘风险溢价系统性上移
- **石油人民币 vs 石油美元**博弈结构性强化
- 全球供应链多元化（替代海峡通道）加速
- 海湾国家加大对核安全/水安全/粮食安全投入
- 欧洲能源转型可能再次提速
