# A股牛市跟踪（2024-09 起）

> 本页跟踪 924 起这一轮 A 股牛市的本质、运行节奏、空间预判与退出框架。**不是预测每周涨跌的择时页，是定位"这是什么牛、能走多远、什么时候该减仓"的战略框架页**。

| 指标 | 数值 | 备注 |

## 二、本质判断（最关键的一层）

**这不是传统牛市，是 BSR 资产荒下的"中国式资产再定价"**。理解这一点决定后续所有判断。

### 2.1 四个前所未有的特征

1. **首个发生在 BSR（资产负债表衰退）中的牛市** — 历史无先例，日本 1990 后从未出现
2. **首个非散户驱动的牛市** — 国家队 + 险资 + 养老金 + 被动 ETF 是主力
3. **首个由"利率塌陷"逼出的牛市** — 10 年国债从 3.0% 跌到 1.5%，存款利率 < 1%
4. **首个被政策刻意"压斜率"的牛市** — "防大起大落"是国家意志，融资保证金 2026-01 主动收紧

**含义**：传统牛熊周期框架不适用。这不是"经济复苏 → 盈利兑现 → 估值扩张"型牛市，而是**"资产荒 + 流动性堰塞湖 + 政策意志"三轮驱动**。

### 2.2 三轮驱动的真实构成

| 驱动 | 性质 | 持续性 |
|---|---|---|
| **政策牛** | 924 政策反转 + "稳中求进、防大起大落" | 强而持久（至少到 2026 末） |
| **流动性牛** | 险资/养老金 2 万亿+增量 + 居民储蓄堰塞湖（174 万亿） | 慢但持续，趋势性 |
| **结构产业牛**（AI/创新药/半导体） | 全球产业革命 + 国产替代 | 最强劲，独立于宏观 |
| **盈利牛** | **缺位** — 全 A 业绩仍负增长 | 最早 2027 年 |
| **居民资产负债表牛** | **缺位** — CCI < 100 | 2027-2028 才有可能 |

## 三、政策为什么必须拉二级市场（深层逻辑）

这是大多数人没追问的问题。答案决定政策托底的"定力强度"。

### 3.1 核心命题

**"消费 + 科技驱动"的经济范式转换，必须配套强大的资本市场——银行体系做不了**。

### 3.2 三层目标叠加

**层次 1: 防御性目标 — 守住金融稳定底线**
- 防止房地产硬着陆引发系统性金融风险
- 防止居民资产负债表螺旋式恶化（房价跌 → 财富缩 → 消费停 → 企业死 → 失业 → 房价再跌）
- A 股稳住 → 居民非房资产端有锚 → 螺旋打断
- **现状：基本完成**

**层次 2: 进攻性目标 — 用财富效应启动内需**
- 174 万亿存款 + 慢牛 → 居民部分财富从存款迁移到股市/基金
- 财富效应 → 信心边际改善 → 消费意愿回升
- **现状：进行中** — Q1 储蓄少增 1.54 万亿是初步信号，但 CCI < 100 说明远未到位

**层次 3: 战略性目标 — 完成增长模型切换（最大的赌注）**
- 从"投资 + 地产 + 出口"驱动 → 切换到"消费 + 科技 + 内循环"驱动
- A 股牛市是这个切换的"配套基础设施"，不是目的
- 时间维度：5-10 年

### 3.3 为什么必须靠二级市场（科技端 + 消费端 + 分配端三重必要性）

**科技端：股权融资生态**
- 科技企业 = 高失败率 + 长周期 + 轻资产 → 银行贷款无法承接
- 字节、宁德、华为、比亚迪没有一家靠银行贷款做大
- 美国 NVIDIA/Apple/Tesla 全部靠 NASDAQ 估值和股权融资
- **传导链**：二级市场活跃 → 一级退出顺畅 → VC 敢投早期 → 创业者拿到钱 → 科技产业崛起
- 2018-2023 港股/A 股科技下跌期，VC 募资同步腰斩 → 大量初创死于无米下炊

**消费端：财富效应载体**
- 消费要起来 → 居民敢花钱 → 安全感 → 来自三条腿（收入/储蓄/资产财富）
- 房产（占居民资产 60%）正在塌
- 储蓄利率 1%，对冲不了通胀
- **股权是唯一可以撑起来的腿**
- 美国 2008 后路径：QE → 股市涨 → 财富效应 → 消费回升

**分配端：替代房产的"全民财富参与机制"**
- 过去 20 年中国最重要的财富分配机制是房地产
- 房地产时代终结后，国家不能让全民失去"参与红利分配"的渠道
- 新机制：央国企分红 + 公募 ETF + 险资养老金 + 个人养老金账户 → 全民通过股权分享国有资产收益
- 类似美国 1980s 后 401k 普及推动的"全民持股文化"

### 3.4 为什么不能用其他工具替代

| 替代方案 | 为什么不行 |
|---|---|
| 降准降息 | BSR 下没人借钱，流动性堆积空转 |
| 发消费券 | 一次性脉冲，3 个月效应消退，改不了财富预期 |
| 企业减税 | 企业不缺钱缺订单，减税不会扩产 |
| 基建大水漫灌 | 边际效用递减，加剧地方债 |
| 房地产救市 | 救它等于让旧周期延续，无法切换新模型 |
| 直接发钱 | BSR 心理 → 大概率被存起来 |

**只有"拉二级市场"能同时打通三件事：财富效应 + 股权融资 + 财富分配**。这就是政策定力如此之强的根本原因。

## 四、板块扩散逻辑（不是切换是扩散）

### 4.1 当前格局（2026-05）

主战场仍在科技 + 周期金属（5-6 数据：电子主力净流入 +52.7 亿、有色 +42.4 亿）。但十大券商 2026 策略共识已是"科技 + 红利 + 消费"均衡牛，工业向消费切换大概率发生在 **2026 H2**。

### 4.2 五个驱动力推动扩散

1. **估值地心引力**：科技部分赛道 PE 已在历史 80%+ 分位，赔率下降
2. **险资/养老金配置约束**：2 万亿增量必须配红利（IRR 5-6% 不能买高估值科技）
3. **PPI 转正传导**：2026-03 PPI +0.5% 终结 28 个月通缩，6-9 个月后 CPI 跟随，利好下游消费品
4. **政策意图均衡**：科技（新动能）+ 消费/红利（财富分配）缺一不可
5. **业绩低基数翻正**：2026 Q2-Q3 同比转正几乎确定（技术性，非真复苏）

### 4.3 警惕"伪轮动"陷阱（关键洞察）

| 真轮动 | 伪轮动 |
|---|---|
| **驱动**：基本面共振（盈利+预期+资金） | **驱动**：单纯资金从高估值流向低估值 |
| **持续性**：6-12 个月 | **持续性**：3-6 个月 |
| **形态**：阶梯式上行，回调不破前低 | **形态**：急涨后阴跌，估值修复一波就走 |
| **历史**：2017 消费白马、2020 消费医药 | **历史**：2019 地产、2023 中特估 |

**这一轮消费/医药的轮动大概率是"伪轮动 + 估值修复"**：
- BSR 约束消费基本面真复苏要等 2027+
- 居民信心未爆发
- 老龄化压制传统消费天花板
- "低基数翻正"是数字游戏

**含义**：消费补涨可期，但赚的是估值修复的钱，不是戴维斯双击的钱，不要 all in。

### 4.4 扩散的真实路径

| 阶段 | 时间 | 主线 | 资金特征 |
|---|---|---|---|
| **A（已完成）** | 2024-09 至 2025 末 | 小盘/题材/AI/低位修复 | 散户 + 游资 |
| **B（当前）** | 2026 H1 | 科技龙头 + 红利双主线 | 险资 + 公募 + 国家队 |
| **C（预计）** | 2026 H2 - 2027 H1 | 科技 + 红利持续 + 消费补涨 + 银行轮动 | 增量扩散，板块快速轮动 |
| **D（最终）** | 2027 H2 后 | 真伪分化期 | 取决于 BSR 修复成败 |

## 五、空间与时间预判

### 5.1 直接答案

**中性情景：上证 5,500-6,500，时间 24-36 个月，可能见顶在 2028 H1 附近**

### 5.2 物理上限四维度推导

| 维度 | 推导 | 上限 |
|---|---|---|
| 估值天花板 | 沪深 300 PE 14x → 18x = +28% | 上证 ~5,350 |
| 估值×盈利双因素 | PE 17x × EPS+15% = 中性 / PE 19x × EPS+20% = 乐观 | 5,480-6,360 |
| 资金流入极限 | 174 万亿存款 5-10% 迁移 = 8.7-17.4 万亿 / 流通市值 80 万亿 → +10-22% | 4,800-5,200 |
| 政策意图 | 5-10 年慢牛 + 涨幅 100-150%（从 2,689 起） | 5,400-6,700 |

**四个维度交叉验证，中性顶部区间收敛在 5,500-6,500**。

### 5.3 三档情景（含赌注定义）

| 情景 | 概率 | 上证目标 | 时间 | 范式转换状态 |
|---|---|---|---|---|
| **悲观** | 25% | 4,800-5,200 | 12-18 月见顶 | 范式转换失败 / 财富效应证伪 |
| **中性** | 45% | 5,500-6,500 | 24-36 月见顶（2028 H1） | 范式转换部分成功 / 慢牛见顶 |
| **乐观** | 30% | 6,500-8,000+ | 5-10 年长牛（到 2030+） | 范式转换全面成功 / 美式长牛 |

### 5.4 各情景的触发条件

**悲观（2027 前任一出现）**：
- CCI 持续 < 95，居民信心修复证伪
- 房价二次下行，财富效应被对冲
- 美联储重启加息，资金外流
- 中美脱钩升级到金融领域
- 政策定力崩塌（被迫放水重蹈 2009/2015 覆辙）
- **剧本**：上证冲到 5,000 附近见顶 → 阴跌到 3,800-4,000 → 进入日本式 5-10 年高位震荡

**中性**：
- 范式转换部分成功（科技起来了，消费没真复苏）
- BSR 修复一半（企业部分复苏，居民信心边际改善）
- **剧本**：上证 5,500-6,500 见顶 → 阶梯式回调 → 进入"准日本 + 准美国"混合形态

**乐观（2027-2028 全部满足）**：
- BSR 完成修复，CCI 重回 100+，房价企稳
- 消费/科技/制造三引擎共振
- 资本市场制度性升级（注册制完善 + 退市常态化 + 全民持股文化）
- 美元信用走弱 + 中国资产相对吸引力上升
- **剧本**：类似美国 2009-2020 长牛 → 上证 6,500-8,000 是中段，真正高点可能在 2030 后

## 六、五个反共识洞察（最容易被忽略的判断）

### 6.1 "牛市不结束" ≠ "还能持续上涨"

资产荒牛市最常见的中后期形态是"高位横盘"，不是继续上涨。日本 1990s、欧洲 2010s 的指数 10 年没新高。**不结束不等于值得持有**——5 年后回头看，可能跑输其他选项 30-50%。

### 6.2 不要用估值判断顶部，用"政策意图变化"判断

经典顶部信号（PE 高、散户疯狂、券商加杠杆）在这轮**全部失效或滞后**。真正的"政策见顶信号"：
- 监管开始警惕泡沫
- 印花税上调 / 加强减持监管 / 主动放开 IPO 供给
- 国家队从买方切换到中性
- **当前：政策仍在加码托底，顶部远未到**

### 6.3 这次"走多远"取决于一件事：居民存款是否真的搬家

- 174 万亿存款是这盘棋的"弹药库"
- 迁移 5%（8.7 万亿）→ 中性情景
- 迁移 15%（26 万亿）→ 乐观情景
- 不迁移 → 悲观情景
- **当前：Q1 仅"少增 1.54 万亿"，远未到搬家阶段**

### 6.4 顶部不会以经典方式出现 — 警惕"温水煮青蛙"

- 这轮顶部大概率不是 2007/2015 的"疯牛崩盘"
- 更像"日本 1989"或"美国 2000 互联网泡沫"——表面繁荣，但分化严重
- **真正的危险不是"指数跌"，是"指数不跌但你的持仓持续跑输"**
- 退出信号不是"暴跌"，是"赚钱效应消失 + 板块轮动加速 + 高股息持续跑赢"

### 6.5 散户全面入场不是顶部信号，是政策目标

跟 2007/2015 完全相反。以前散户狂热是危险信号，**这次散户入场就是政策的目的**（全民持股 + 财富分配）。所以"开户数飙升"反而是中段信号，不是末段信号。

## 七、退出框架（不看估值看政策意图）

### 7.1 真正的退出信号（按重要性排序）

1. **政策意图变化**：监管警惕泡沫 / 上调印花税 / 加强减持 / 放开 IPO 供给
2. **赚钱效应消失**：板块轮动加速到 < 1 个月，没有持续主线
3. **国家队从买方变中性**：被动 ETF 流入停滞或转出
4. **沪深 300 PE > 18x**：估值进入历史 80%+ 分位
5. **公募月度发行 > 5,000 亿**：散户狂热的滞后信号
6. **居民开户数月度 YoY > 200%**：疯牛阶段
7. **巴菲特指标（市值/GDP）> 100%**：进入泡沫区

**任一信号触发开始警惕，2 个以上同时出现执行减仓**。

### 7.2 验证 / 证伪信号（基本面端）

| 信号 | 含义 | 当前状态 |
|---|---|---|
| 银行板块持续 2 月跑赢科技 | 险资真买盘启动 | 部分出现 |
| 消费白马 PE 修复到历史 50%+ 分位 | 估值修复完成 | 未到 |
| 沪深 300 跑赢创业板 2 个季度 | 风格切换确认 | 未到 |
| PPI 持续 3 月正区间 + CPI 单月 ≥ 1.5% | 基本面传导信号 | PPI 刚转正，CPI 未起 |
| M1 持续 3 月 ≥ 8% + 剪刀差转正 | BSR 进入修复期 | 未到（M1 5.1%、剪刀差 -3.4pct） |
| CCI 持续 ≥ 100 | 信心侧修复 | 远未到（< 100） |
| 住户存款同比走平/负增长 | 存款搬家真信号 | 仅"少增"，未到 |
| 房价同比转正 | 财富效应基础修复 | 未到 |

**当前 8 个信号几乎全未确认**，所以扩散刚萌芽，远未到顶。

## 八、持仓应对节奏表

### 8.1 按指数位置的分阶段应对

| 阶段 | 上证位置 | 动作 |
|---|---|---|
| **当前 → 5,000** | 4,180 → 5,000 | 维持高仓位，扩散到红利 + 消费，科技降弹性 |
| **5,000 → 5,500** | 进入估值中性区 | 开始减弹性赛道（高估值科技、概念股），保留龙头 + 红利 |
| **5,500 → 6,500** | 接近中性顶部区 | 降到中等仓位（40-60%），只保留高股息 + 逻辑确定的科技龙头 |
| **6,500+** | 进入"赌乐观情景"区 | 极低仓位 + 期权对冲，等政策意图变化或 BSR 证伪信号 |

### 8.2 按板块的差异化应对

| 仓位类型 | 当前应对 | 逻辑 |
|---|---|---|
| 科技龙头（AI/半导体/创新药） | 不清仓但停止加仓，回调时择优换仓 | 主线还在但赔率下降 |
| 科技高弹性赛道（机器人/题材） | 减仓锁定收益 | 估值过高，扩散下相对吸引力下降 |
| **红利**（银行/电力/电信/煤炭） | **持续加仓底仓** | 险资主力 + 资产荒 + 财富分配主线，不是轮动对象是趋势 |
| 消费白马（白酒/家电/医药龙头） | 历史 30% 分位以下分批建仓，不重仓 | 估值修复有空间，但赌伪轮动不要 all in |
| 周期金属（有色/黑色） | 高位减仓 | 估值已偏高 |
| **银行** | **持续加仓** | 险资+红利+净息差企稳+估值仍便宜，最具确定性的扩散方向 |
| 保险/券商 | 等机会 | 牛市受益但当前估值已抬升 |

### 8.3 核心原则

**"分阶段调结构"，不是"持有到顶部"**。BSR 牛市的赚钱逻辑是动态调结构，不是静态持有。

## 九、关键风险（按概率排序）

| 风险 | 等级 | 触发条件 | 影响 |
|---|---|---|---|
| **"温水高位"机会成本** | 高 | 牛市 1-2 年后维持但不涨 | 5 年跑输其他选项 30-50% |
| **财富效应转化失败** | 高 | 2027 年 CCI 仍 < 100 / 储蓄未真正搬家 | 范式切换失败，日本病 |
| **房地产二次下行** | 中高 | 房价继续阴跌 / 销售再下台阶 | 财富效应被对冲 |
| **政策定力崩塌** | 中 | 经济压力下被迫"大水漫灌" | 短期烟花 + 长期更深 BSR |
| **美国脱钩升级** | 中 | 关税 / 科技封锁 / 金融脱钩 | 出口受阻 + 资本外流 |
| **美联储重启加息** | 低 | 全球通胀复燃 + 美元再强势 | 资产荒逻辑瓦解（唯一会触发经典熊市的剧本） |
| **人口拐点压制** | 长期 | 老龄化 + 出生率下行加速 | 内需天花板压制 |

**经典 2015 式杠杆崩盘的概率被刻意压到极低**（"防大起大落"+ 主动收紧融资保证金）—— 这次政策不会重蹈覆辙。

## 十、关键观察节点

| 节点 | 观察什么 | 含义 |
|---|---|---|
| **2026-07 政治局会议** | "降准降息"是否回归 | 货币政策转向信号 |
| **2026-Q2-Q3 业绩季** | 全 A 业绩同比是否翻正 | 低基数效应验证 |
| **2026-H2** | 工业 → 消费板块切换是否启动 | 扩散行情主升 |
| **2027-H1** | CCI 是否突破 100 + 储蓄是否同比走平 | BSR 修复关键判断点 |
| **2027-H2** | 全 A 盈利是否进入恢复性增长 | 中性 vs 乐观情景分水岭 |
| **2028-H1** | 中性顶部区间是否到来 | 关键减仓窗口 |

## 十一、一句话总结

**这轮牛市能走多远，本质上不是市场决定的，是"中国能否完成 BSR 退出 + 范式转换"决定的**：
- 成功 → 上证 6,500-8,000+，持续 5-10 年，国运级行情
- 部分成功 → 上证 5,500-6,500，2028 见顶，长期高位震荡
- 失败 → 上证 5,000 是顶，然后日本化 10 年

**当前我倾向"中性 45% + 乐观 30% = 75% 概率最终走到 5,500+"，但中间会有 2-3 次 15-20% 回调**。

最重要的判断不是"涨多高"，是: **不要在 5,000-5,500 区间因为"还能涨"放弃减仓决策——这个区间是"中性顶部"和"乐观中段"的分水岭，前者意味着卖了对，后者意味着卖了错过 30%+。这个判断要在 2027 H1 基于 BSR 修复信号才能做出**。

在那之前，**保持仓位、扩散结构、不做单边赌注**，是这轮牛市 90% 时间里的最优策略。
