# 半导体周期追踪

> 追踪AI投资热潮驱动的半导体周期转折信号。本页面用于定期复盘，判断周期位置。

## 周期定位

**当前阶段：上行中后期（第12个季度）**

- 上一轮底部：2023年4月（YoY转正）
- 传统周期长度：3-4年（上行~2年+下行~2年）
- 本轮特殊性：AI结构性需求拉长了上行期，但历史规律（Capex增速>56%后次年必减速）仍未被证伪
- 最可能的转折时点：**2026Q4-2027Q2**

## 五个领先指标

按信号出现的先后顺序排列。信号1最早出现，信号5最晚出现。

### 信号1：GPU交货周期

**原理：** 从"有钱买不到"到"下单就能拿"是供需逆转的第一步。交货周期缩短 → 供给追上需求 → 接下来就是过剩。

| 日期 | B200交期 | B300交期 | 积压量 | 状态 |
|------|---------|---------|--------|------|
| 2025Q4 | 12-24周 | — | 360万台 | 🟢 极度紧张 |
| 2026Q1 | 8-16周 | 12-20周 | 高位 | ⚠️ 微松 |
| | | | | |

**转折阈值：** B200交期<6周，B300<8周，积压清零

### 信号2：存储价格

**原理：** 存储是半导体最敏感的温度计。价格拐点通常领先行业周期3-6个月。

| 日期 | DRAM合约价QoQ | HBM价格趋势 | 状态 |
|------|-------------|------------|------|
| 2025Q4 | +40% | 持续上涨 | 🟢 景气 |
| 2026Q1 | +55-60%(服务器>60%) | 仍涨 | ⚠️ 可能见顶 |
| 2026Q2 | 续涨(mobile领涨) | HBM3e 走强，DDR5 缺口拉大 | ⚠️ **拐点延后** |

**转折阈值：** DRAM合约价连续2个月环比下跌；HBM出现降价

**预测：** Q2 DRAM 续涨（供应商向服务器/HBM 倾斜，mobile 抢补库存），TrendForce 原预期"Q3 -14%"被推迟；HBM3e 因 DDR5 挤占产能反而涨价。**拐点预计推迟至 2026Q3-Q4**

### 信号3：云厂商Capex指引

**原理：** 四大云厂商（Amazon/Google/Meta/Microsoft）是AI芯片最终买家，Capex指引决定了NVIDIA/TSMC的订单。

| 日期 | 四大合计Capex | YoY增速 | 关键变化 | 状态 |
|------|-------------|---------|---------|------|
| 2025 | ~$365B | — | 全面加速 | 🟢 |
| 2026指引 | ~$570-690B | +56-89% | 仍在加速 | 🟢 但增幅过大 |
| | | | | |

**转折阈值：** 任何一家说"2027年Capex持平或下降"；Alphabet FCF暴跌90%后股东施压

**关键观察点：** 
- Microsoft增速最慢(+12%)，最可能第一个调低
- Alphabet FCF从$73.3B暴跌至$8.2B，不可持续
- **每季财报后更新此行**

### 信号4：TSMC产能利用率

**原理：** TSMC是全球AI芯片的唯一制造商（70%份额），其利用率是行业健康度的金标准。

| 日期 | 先进制程利用率 | CoWoS状态 | 季度营收增速 | 状态 |
|------|-------------|----------|------------|------|
| 2025Q4 | >95% | 满配到2027中 | +35% | 🟢 |
| 2026Q1 | ~100%(先进制程满载) | 满配到 2027 全年 | **+35%**（毛利率 66.2% 历史新高/净利+58%） | 🟢 **走强** |
| 2026-03 | — | — | 单月营收 NT$415 亿 **+45.2% YoY**（史上最强单月） | 🟢 |
| 2026-04 | — | — | 单月营收 $11.55B（创纪录） | 🟢 |

**转折阈值：** 先进制程利用率<85%；CoWoS出现空余产能；季度营收增速<15%

**当前：** CoWoS 满配延长至 2027 全年（Bernstein 预估 2026 末达 12.5 万片/月，接近翻倍），2026 Capex $520-560 亿 +30% YoY 历史新高 → 信号不仅未松动反而**更加偏绿**，预计 2027H2-2028 才可能缓解。

### 信号5：NVIDIA指引

**原理：** NVIDIA是本轮周期的晴雨表，也是最滞后的信号。等到NVIDIA miss再跑已经晚了。

| 日期 | 季度营收 | 数据中心增速 | 库存天数 | 指引vs预期 | 状态 |
|------|---------|------------|---------|-----------|------|
| FY2025 | $2057亿 | +65% | — | 持续beat | 🟢 |
| | | | | | |

**转折阈值：** 数据中心增速<30%；库存天数上升；季度指引低于市场预期

## 可能的触发事件

### 高概率

| 事件 | 时间窗口 | 机制 | 影响 |
|------|---------|------|------|
| 存储价格环比转跌 | 2026Q2-Q3 | DRAM/HBM合约价拐点 → 存储股重估 | 存储/设备股先跌 |
| 云厂商砍Capex | 2026Q4-2027Q2 | FCF不可持续 → 股东/董事会施压 | 全链条冲击 |
| AI ROI证伪 | 2026H2-2027 | 企业AI投资回报不及预期 → 削减预算 | 杀估值逻辑 |

### 中概率

| 事件 | 机制 | 影响 |
|------|------|------|
| "DeepSeek 2.0"效率突破 | 证明不需要这么多GPU | 杀需求逻辑 |
| NVIDIA指引miss | 单季miss → 恐慌抛售 | SOX可能一周跌15-20% |
| 宏观衰退 | 美联储不降息/硬着陆 | 系统性风险 |

### 低概率高冲击

| 事件 | 冲击 |
|------|------|
| 台海紧张 → TSMC停产 | 全球芯片链断裂 |
| 主要国家限制AI训练规模 | 需求骤降 |
