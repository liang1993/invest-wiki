# 人形机器人

## 行业概述

人形机器人是 2025-2026 年全球**真量产临界点**的硬科技赛道。2025 年全球出货约 **1.3 万台**（已知厂商交付汇总口径，证券时报 2026-01-09），同比 +500% 量级；TrendForce 预测 2026 年中国产量同比 +94%，全年下半年进入商业化关键阶段。

行业本质：**电驱+精密机械** 的工程问题，AI（大脑）是输入而非主战场——价值链 70-80% 在精密零部件（丝杠/电机/减速器/传感器），中国制造在此环节具备比较优势。

### 阶段判断

类比意义最近的对标是 **2014-2015 年的特斯拉链** 与 **2010 年的 iPhone 链**：
- 量产真在发生（特斯拉 V3 投产临近、智元/宇树万台级出货、Atlas 产线落地）
- 但 2026 全行业总收入仅约 150-500 亿元（5-10 万台 × 单价 30-50 万元），切到核心零部件厂头上单家增量收入 5-20 亿
- 当前估值已 price in 2026-2027 高速增长，**叙事-业绩剪刀差** 是最大风险

### 应用场景成色（2025 估算）

> 这是判断"真增长 vs 估值泡沫"的核心指标——同样 1 台机器人在不同场景的可持续性差距巨大。

| 场景 | 占比估算 | 商业可持续性 | 评价 |
|---|---|---|---|
| 科研教育 / 数据采集 | 40-50% | 弱 | 高校/地方政府数据训练中心，本质是**财政补贴** |
| 文娱商演 / 春晚 / 接待 | ~20% | 弱 | 一次性消费，复购低 |
| 导览导购 / 政务大厅 | ~10% | 中 | 替代效应有限，更像营销噱头 |
| **物流分拣 / 仓储** | ~10% | **强** | 京东、菜鸟试点，效率账可算 |
| **工业制造**（汽车/3C） | ~10-15% | **强** | 现代、富临精工、BMW、特斯拉自家工厂 |
| 家庭服务 | <1% | 暂未启动 | 至少 5 年外 |

**结论**：2025 的 1.3 万台里，约 20-25% 是真正"算得清账"的工业/物流场景（约 3,000-4,000 台）。**2026 关键不是出货总量，而是工业场景占比能否从 25% 提到 40%+**——这是从"主题"切换到"业绩"的真正分水岭。

## 产业链与 BOM

### 三条技术路线

| 路线 | 代表 | 特点 |
|---|---|---|
| **电驱 + 丝杠 + 总成**（主流） | Optimus、Figure、Atlas、智元 | 单台 BOM 5-8 万美元，规模化降本路径清晰，A 股零部件链 90%+ 跟这条 |
| **轻量化感知** | 宇树 H1/H2 | 强调运动性能（10m/s 跑步），偏爆款单品逻辑 |
| **液压方案**（已淘汰） | 老 Atlas | 已被电驱方案取代 |

### BOM 价值占比（特斯拉 Optimus 口径）

| 环节 | 价值占比 | 备注 |
|---|---|---|
| **行星滚柱丝杠** | 19-29% | 单价 ~1500 元/个，用量大 |
| **无框力矩电机** | 15% | 起步阶段，国产化率低 |
| **谐波减速器** | 7-13% | 用于小臂、腕部、手部 |
| **力 / 力矩传感器** | 7-11% | 国产渗透低 |
| **空心杯电机** | 4-8% | 进口为主 |
| **编码器** | 4-5% | — |
| **其他**（结构件/轴承/电池/MCU/AI 芯片） | 30-40% | 分散 |

> 价值占比合计 ~70-80% 都在**精密机械**——人形机器人本质是"特斯拉 Model 链 2.0"，AI（NVIDIA、特斯拉自研芯片）只占小头。

## 核心公司（A 股 / 港股）

### 高确定性赛道

- [绿的谐波](../stocks/tech-hardware/绿的谐波.md) — 谐波减速器国产第一（国内份额 22%，国产 TOP4 合计 64%），2025 营收 5.69 亿（+46.86%），归母 1.25 亿（+122%）；2026E 人形相关收入有望 >3 亿，占比从 ~13% 跳到 30%+；估值偏高（PE >150x）
- **双环传动（002472）** — RV 减速器国内份额 50%+，工业机器人主用；人形机器人多数用谐波+丝杠，所以人形敞口比绿的小；估值 ~30x PE
- **三花智控（002050）** — 特斯拉 Optimus 执行器总成核心供应商，与拓普已在墨西哥设厂；本业空调零部件 ~20x，机器人溢价拉到 30x
- **拓普集团（601689）** — 主业 NVH/底盘，人形布局线性执行器+灵巧手，PE ~25x

### 高弹性 / 主题股

- **鸣志电器（603728）** — 空心杯电机国内龙头，BOM 占比 4-8%，但 2025 业绩承压
- **北特科技 / 贝斯特 / 震裕** — 行星滚柱丝杠主题股，竞争激烈，多家亏损
- **柯力传感 / 东华测试** — 力/力矩传感器，国产渗透低
- **兆威机电 / 拓普** — 灵巧手相关
- 工业机器人本体：[北方华创](../stocks/tech-hardware/北方华创.md)（半导体设备龙头，人形敞口低，但工控生态相关）、埃斯顿（A股工业机器人份额第一 10.5%）

### 港股本体厂

- **宇树科技（09880.HK）** — 中国第一只人形机器人本体上市公司，2025 出货 4,200-5,500 台，运动性能全球领先，但本体厂利润率薄、估值用 PSR

### 美股 / 其他

- 特斯拉（TSLA）— Optimus 自研，整车厂内部消化路径
- Figure / 1X / Apptronik — 未上市
- Boston Dynamics（属于现代汽车）— 未独立上市

## 价值链利润分布判断

| 环节 | 护城河强度 | 理由 |
|---|---|---|
| 谐波减速器（绿的） | **强** | 国产 22%/全球第二（仅次哈默纳科），精密制造壁垒+客户认证周期长 |
| RV 减速器（双环） | 中强 | 50% 国内份额，但下游议价力强 |
| 平台型工控（汇川） | 中强 | 多产品线交叉销售，但人形敞口小 |
| **行星滚柱丝杠** | **弱** | 多家在卷，价格战必然，毛利会被打下来 |
| 总成/特斯拉链（三花、拓普） | 中 | 强绑定单一大客户，"成也萧何"风险 |
| 整机（宇树/智元/小米/优必选） | **弱** | 本体厂集中度低、毛利薄，类似汽车整车 |

**核心判断：人形机器人的钱大概率被精密零部件赚走，整机厂吃辛苦钱。**

## 风险因素

1. **特斯拉 V3 跳票**（高概率，已发生 2 季度）→ 板块估值杀 30-50%
2. **工业场景渗透不及预期**（中等概率）→ 业绩兑现年从 2026 推到 2027
3. **本体厂垂直整合**（持续发生）→ 现代 Atlas 内部消化、特斯拉自研零部件、宇树 90% 零部件自产，**第三方供应商蛋糕变小**
4. **AI 大模型瓶颈** → 具身智能"小脑"突破不了，行业回到"花瓶机器人"
5. **海外科技封锁** → 减速器、丝杠加工设备（德日精机）、AI 芯片（英伟达）受制于人
6. **A 股资金面松紧** → 主题股对流动性高度敏感，估值压缩往往比基本面变化快
