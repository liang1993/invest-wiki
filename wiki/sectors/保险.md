# 保险行业

## 行业总览

中国保险业是**金融三支柱（银行/证券/保险）中渗透率最低、长期成长空间最大的子行业**。2025 年总保费 6.12 万亿（+7.43%），保险密度（人均保费）约 5,150 元，**保险深度（保费/GDP）4.3%**，距离全球均值 7%、发达国家 10%+ 仍有显著差距。

商业模式两条主线：
1. **承保利润**（保费收入 - 赔付 - 费用）—— 财险公司核心驱动
2. **投资利差**（投资收益 - 准备金成本）—— 寿险公司核心驱动

→ **保险股 = 保费增长 × 综合成本率（财险）× 投资收益率（寿险）三因子叠加**

## 行业子板块特征

### 1. 寿险（3.56 万亿 / 行业最大子板块）

**商业模式**：长期保单 + 投资端利差
- 长期主营业务（10-30 年保单）
- 核心 KPI：**新业务价值 NBV** + **内含价值 EV** + 综合投资收益率
- 估值方法：**P/EV**（行业标准，类似银行的 PB）

**当前景气位置**：**复苏中段**
- NBV 2022-2023 谷底（行业 -10-20%）→ 2024-2025 拐点（+25-29%）
- 银保渠道是核心增量（平安 +138%、太保 +50%+、新华 +20%+）
- 储蓄型产品占比仍高（70%+），保障型产品占比偏低（20-25%）

**关键变量**：
- **预定利率下调**（2.5% → 2.0% 切换中）：短期刺激抢购，长期压缩负债成本
- **分红险全面切换**（2025-2026 推行）：从固定收益转向浮动收益，转嫁投资风险
- **代理人队伍**：从 2019 年 900 万峰值降至 2024 年 ~300 万，但 NBV/人均产能 +50%+
- **银保渠道占比**：2025 已超 30%，预计 2026 继续提升至 35%+

### 2. 财险（1.76 万亿 / 行业第二子板块）

**商业模式**：短期保单（1 年） + 承保利润 + 投资利差
- 短期主营业务（1 年保单为主）
- 核心 KPI：**综合成本率 COR** = 赔付率 + 费用率
- COR < 100% = 承保盈利；COR > 100% = 承保亏损
- 估值方法：**PE**（业绩稳定）+ PB

**当前景气位置**：**周期改善**
- 2024-2025 行业 COR 从 99-100% 改善到 96-98%
- 车险（占 50%）：政策性"报行合一"压低费用率
- 非车险（占 50%）：科技 / 健康 / 责任险高增

**关键变量**：
- **车险改革**：综改后费用率压低，但保费增长承压
- **新能源车险**：定价博弈，赔付率高（120%+）但渗透率提升
- **巨灾风险**：极端天气频率上升 → 2024 河南暴雨等单一事件影响 COR
- **非车增量**：健康险/责任险/农险 +15-25%

### 3. 健康险（合并 9,973 亿，接近万亿）

**商业模式**：医疗保障 + 重疾保障
- 短期医疗险（百万医疗）+ 长期重疾险
- 核心 KPI：**赔付率**（医疗险 70-85% / 重疾 30-50%）

**当前景气位置**：**长期上行通道**
- 老龄化驱动（2030 年 65 岁以上人口 3 亿）
- 医保报销不足驱动商保需求
- 2025 +8.5%，预计 2026-2030 +10-15%/年

### 三阶段周期框架

### 复苏驱动力

1. **储蓄需求旺盛**：居民住户存款 174 万亿，房产 / 银行理财净值化后**保险成为新避风港**
2. **预定利率下调红利**：2.5%→2.0% 抢购潮 + 长期负债成本下降
3. **代理人队伍质量提升**：清虚 + 数字化赋能 + NBV/人均产能 +50%
4. **银保渠道爆发**：低成本获客，2026 占比预计 35%+
5. **股市企稳**：投资端 6.49 万亿组合是巨大杠杆
6. **健康险长期红利**：老龄化 + 医保替代

### 复苏阻力

1. **利率长期下行**：10Y 国债 1.74% → 寿险公司投资端持续承压
2. **房地产敞口**：保险资金端房地产配置 4-6% 仍需消化
3. **分红险切换阵痛**：客户接受度待考验
4. **车险保费 -0.6%**：行业承压（费用率压低 + 保费下行）
5. **新会计准则（IFRS17/IFRS9）影响**：利润波动放大

## 估值方法（保险特有）

### 1. P/EV（保险行业主估值，必学）

**EV（内含价值）= 调整后净资产 + 现有有效业务价值（VIF）**

- EV 是寿险公司的"实际净资产"概念，远比账面 BVPS 更能反映真实价值
- 历史 P/EV 区间：0.5 - 1.5x（牛市顶 vs 熊市底）
- 各公司当前 P/EV：
  - 中国人寿 ~0.55x（折价最深）
  - 中国太保 ~0.55x
  - 中国平安 ~0.70x
  - 新华保险 ~0.50x
  - **友邦保险 ~1.5x**（行业品质天花板）

### 2. PB（金融行业基础）

- A 股险企 PB 1.0-1.8x
- ROE 普遍 11-14%
- 合理 PB ≈ ROE × 调整系数

### 3. PE（财险更适用）

- 财险业绩稳定 → PE 10-12x 合理
- 寿险投资端波动大 → PE 不稳定（仅作参考）

### 4. SOTP（综合金融适用）

- 适用于平安、太保等多业务险企
- 平安：寿险 P/EV + 财险 PE + 平银 58% 持股 + 资管/科技
- 太保：太保寿险 P/EV + 太保财险 PE

### 5. DDM（高分红险企）

- 中国财险、人保等高股息标的
- 股息率 5%+ 的稳定分红 + 永续增长 g 5%
- 折现率 r 8%

## 投资逻辑

### 看多保险行业（中长期）

1. **保险渗透率长期上行空间**：保险深度 4.3% vs 全球均值 7% / 发达国家 10%+
2. **居民资产配置切换**：房产塌陷 + 存款负利率 → 保险成为新避风港

5. **健康险长期红利**：老龄化 + 商保替代医保
6. **估值历史中位偏低**：除人寿外，平安/太保/新华/人保 P/EV 均 < 0.7x

1. **利率持续下行**：10Y 国债跌破 1.5% 是全行业 ROE 大跌的强信号
2. **股市持续低迷**：投资端 6.49 万亿组合是双刃剑
3. **车险价格战**：持续压低 COR
4. **房地产风险二次释放**：保险资金端配置 4-6% 是历史包袱
5. **监管收紧**：偿付能力监管 + 资本补充压力

## 行业政策环境

**2025-2026 关键政策**：

1. **预定利率下调（2025-2026 完成）**：
   - 2.5% → 2.0%（普通型）
   - 影响：短期刺激抢购，长期压缩负债成本利好寿险公司
2. **分红险全面切换（2025-2026 推行）**：
   - 从固定收益转向浮动收益
   - 转嫁投资风险给客户，但短期销售难度大
3. **车险"报行合一"（2024 起）**：
   - 严格控制费用率
   - 短期保费承压，中期 COR 改善
4. **新会计准则（IFRS17 + IFRS9）落地（2024-2025）**：
   - 利润表波动加大
   - 寿险公司利润前置（NBV 直接释放至当期利润）
5. **偿付能力监管（C-ROSS II）**：
   - 资本要求提升
   - 中小险企承压，头部受益
6. **健康中国 2030 + 老龄化政策**：
   - 健康险长期红利
   - 商保对接医保结算

→ **保险内部**：寿险弹性 > 财险，但财险 COR 稳定 + 高股息

- 仓位：5-10%（接受估值贵的代价换确定性）

## 与组合层面的关系

**保险股 vs 银行股**：
- 都是金融板块，但驱动逻辑不同（NBV vs 息差）
- 保险股的"复苏弹性"在 NBV 持续兑现期更强
- 银行股的"估值修复弹性"在 PB 极低位更强（如平银 PB 0.47）
- 不应同时重仓两者（板块集中度过高）

**保险股 vs 高股息标的**：
- 保险股股息率 4-5.5%，与水电（长江电力 3.5%）/ 国有大行（5-7%）相当
- 优势：成长性更强（NBV 复苏 + 健康险长期红利）
- 劣势：周期波动大于水电 / 国有大行
