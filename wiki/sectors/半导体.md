# 半导体

## 行业概述

半导体是全球科技产业的底层基础设施，覆盖计算、通信、汽车、工业、消费电子等几乎所有数字化领域。2025年全球半导体销售额达 **7956亿美元**，同比增长 **26.2%**（WSTS数据），为历史上增速最强的年份之一。2026年预计将达 **9754亿美元**（+22.5%），逼近万亿美元里程碑。

**增长驱动力排序：**
1. **AI基础设施**（数据中心GPU/ASIC/HBM）：2025年计算机芯片segment同比增长60%+，是核心驱动力
2. **存储器超级周期**：HBM需求爆发+传统DRAM供不应求，存储板块2026年预计增长30%+
3. **汽车半导体**：电动化+智能化持续推动单车半导体含量提升，但2025-2026面临DRAM短缺压力
4. **IoT/工业**：复苏较慢，传统芯片领域仍有库存压力和价格战

## 行业周期

### 当前位置：上行周期中后段

- **周期定位：** 半导体行业已进入2026上行周期（upcycle）深处，若按历史16个季度的典型周期计算，营收增速可能在2026年Q2附近见顶
- **月度增速：** YoY增长率已连续近一年保持20%以上
- **库存状态：**
  - 全球库存天数已从2023年的120+天降至2026年初的 **65天**（健康水平）
  - 先进制程/AI芯片：**持续供不应求**，TSMC CoWoS产能、HBM产能均为瓶颈
  - 成熟制程/传统芯片：供应商库存仍偏高，需求恢复较慢，部分领域存在价格战
- **存储器：** DRAM持续供不应求至2026年Q1，存储价格Q1再涨40%+，特斯拉等车企已宣布DRAM短缺影响产能

### 2025-2026景气度展望

| 细分领域 | 景气度 | 展望 |
|------|------|------|
| AI芯片（GPU/ASIC） | 极高 | 供不应求持续至2026年底 |
| HBM存储 | 极高 | SK Hynix 2026年产能全部售罄 |
| 先进制程代工 | 高 | TSMC 3nm/2nm持续扩产 |
| 传统DRAM/NAND | 中高 | 涨价周期，但可能Q2后放缓 |
| 汽车半导体 | 中 | 电动化需求稳健，但DRAM短缺制约 |
| 成熟制程/模拟/功率 | 中低 | 去库存尾声，价格压力仍在 |
| IoT/消费电子 | 低 | 复苏缓慢，竞争加剧 |

## AI对半导体的拉动

### AI芯片市场规模

- **2025年：** 全球AI芯片市场约 **944亿美元**（Precedence Research），同比增长~33%
- **2026年：** 预计达 **1219亿美元**（+29%）
- **2034年：** 预计达9313亿美元（CAGR 28.9%）

### 细分市场

| 类别 | 2025年规模 | 市场格局 |
|------|------|------|
| **GPU** | 占AI芯片市场48.8% | NVIDIA占绝对主导（A100/H100/H200/B200系列） |
| **定制ASIC** | 快速增长，边缘推理78亿美元 | Google TPU、Amazon Trainium、Microsoft Maia、Meta MTIA |
| **HBM** | **350亿美元**（占DRAM 23%） | SK Hynix 62%份额 > Micron > Samsung |

### HBM存储（AI芯片核心配套）

| 指标 | 数据 |
|------|------|
| 2025年HBM市场 | 350亿美元 |
| 2026年HBM市场 | **546亿美元**（+58%，美银预测） |
| 2028年HBM市场 | 1000亿美元（~40% CAGR） |
| SK Hynix市占率 | 62%（出货量）/ 57%（营收） |
| Micron HBM营收 | 季度~20亿美元（年化~80亿美元） |
| Samsung HBM目标 | 2026年市占率恢复至30%+ |

**技术迭代：** HBM4已完成开发（SK Hynix，功耗降40%，速率10Gbps），Micron也已送样HBM4（速率11Gbps），预计2026年下半年进入量产。

### AI对行业结构的影响

- NVIDIA 2025年营收2057亿美元中，数据中心占比超80%，AI几乎重新定义了这家公司
- TSMC约75%营收来自7nm及以下先进制程，AI芯片是核心拉动力
- 到2029年底，AI将驱动全球超过50%的芯片销售额
- AI XPU（含GPU）2025-2029年将从当前水平增长2.3倍至4650亿美元

## 地缘政治：美国对华芯片出口管制

### 管制演进时间线（2022-2026）

| 时间 | 事件 |
|------|------|
| 2022.10 | BIS首次对华先进芯片出口管制（A100/H100级别） |
| 2023.10 | 管制升级，收紧算力阈值，扩大设备限制 |
| 2025.01 | BIS发布"AI扩散框架"（AI Diffusion Rule），限制第三国转运 |
| 2025.05 | AI扩散框架被撤销又重新修订 |
| 2025.07 | **美国取消对华EDA出口限制**（Synopsys、Cadence、Siemens EDA受益） |
| 2025.09 | 特朗普政府扩大管制，延伸至外国关联企业 |
| 2025.12 | 特朗普宣布允许H200向中国批准客户出口 |
| 2026.01 | BIS修订许可政策：H200/MI325X等逐案审批（附安全条件） |
| 2026.04 | **MATCH法案**提出：管制范围从EUV扩大至DUV光刻机、刻蚀机、沉积设备等 |

### MATCH法案要点（2026年4月最新）

- **设备管控升级：** 从EUV光刻机扩展至DUV光刻机、刻蚀机、薄膜沉积、清洗设备等全品类
- **服务禁令：** 禁止美国及关联企业对已售中国设备提供安装、调试、维护、软件升级、远程技术支持
- **指定管控实体：** 中芯国际(SMIC)、华为、长江存储、长鑫存储、华虹集团及其附属公司
- **盟友施压：** 要求荷兰、日本、韩国在150天内全面对齐美国管控，否则启用"外国直接产品规则"（二次制裁），迫使ASML、Tokyo Electron、Samsung等配合
- **战略目标：** 阻止中国在AI、超算等领域实现"弯道超车"，构建美国主导、盟友参与、排除中国的"小院高墙"

### 对行业的影响

- **中国半导体：** 加速国产替代，中芯国际成熟制程扩产，但先进制程（7nm以下）受限
- **设备商：** ASML、Tokyo Electron面临中国市场营收损失风险，ASML已表示中国营收占比将下降
- **Fabless：** NVIDIA被迫开发中国特供版芯片（H20等），但性能受限
- **不确定性：** 特朗普政府政策反复（先收紧后放松EDA管制，先禁后允H200出口），企业合规成本上升

### 关键判断

**短期（6-12 个月）—— Capex 仍在加速，半导体收入端无忧**：
- 全年指引同比 +77%，Q1 实际同比 +72%，斜率仍在
- TSMC CoWoS、HBM、NVIDIA GPU 仍供不应求

**中期信号已分化**：
1. **Q1 环比仅 +4%** — 同比仍高但环比放缓，加速度可能见顶
2. **MSFT $25B / $190B（13%）来自组件涨价** — 被动加投占比明显，capex 总量"虚胖"
3. **Meta 财报后跌 6%** — 市场首次对"capex 上调"给出**负反馈**，开始追问产出
4. **GOOG Cloud +63%、backlog $462B 翻倍** — 唯一让市场满意，capex 与 ARR 兑现度最高
5. **三年 60%+ 增速 → 数学上不可持续**：2027 capex 要再加 $400B+，自由现金流将受压

### 对应 Burry 期权时点

Burry 选 2027/1 到期，对应：
- 2026 年报披露 → 2027 capex 指引出炉
- 若 2027 capex 进入"高位平台"或下修 → 卖方共识 EPS 下修 → SOX 60× PE × 下修 EPS = 双杀
- 这是"capex 见顶"假设最合理的兑现窗口

### 跟踪指标（每季度财报后更新）

1. **四家季度 capex YoY/QoQ** — 同比 + 环比双指标，关注环比是否转负
2. **capex / 营收比** — 已逼近 30%，历史上不可持续的水平
3. **Cloud 收入增速 vs capex 增速** — GOOG 跟得上、META 跟不上 → 分化交易机会
4. **MSFT 组件成本占比** — 区分主动加投 vs 被动涨价
5. **NVDA 数据中心 backlog 与超大规模厂商 capex backlog 的比值** — 领先指标

### 操作含义

- **不必现在就清仓半导体相关**：capex 仍加速，估值杀伤需要触发器
- **NVDA / SOX 60× PE 已是 price perfection**：容错率极低，任何一家 hyperscaler capex 指引下修都会触发回调
- **国产替代受益较小但相对独立**：A 股半导体板块本身估值在 97% 分位（见下文），与美股 SOX 高 β，不是避风港

## 中国半导体国产替代专题（2026.04 深度调研）

> 以下为中国半导体全产业链国产替代的专项调研，覆盖政策资金、各环节进展、卡脖子分析、AI芯片、投资机会六大维度。

### 国产替代全景概览

中国半导体国产替代是当前最具战略意义的产业主题。在美国持续升级出口管制背景下，中国正以"举国体制"推进全产业链自主可控。2025年产业整体国产化率突破**28%**（同比+8pct），半导体设备国产化率跃升至**50%**，科创板128家半导体企业2025年合计营收超**3600亿元**（同比+25%）。

| 环节 | 国产化率（2025） | 核心玩家 | 趋势 |
|------|-----------------|---------|------|
| **芯片设计** | 分化严重 | 海思、寒武纪、海光、韦尔、兆易 | AI芯片爆发，消费芯片复苏 |
| **晶圆制造** | 成熟制程80%+/先进<20% | 中芯国际、华虹 | N+2量产，产能持续扩张 |
| **封测** | 60%+ | 长电、通富微电、华天 | 先进封装（HBM/Chiplet）是增长极 |
| **设备** | 整体50%（差异极大） | 北方华创、中微、拓荆、盛美 | 去胶74%→光刻机1.1%，极度分化 |
| **材料** | 整体32% | 沪硅产业、安集、南大光电、雅克 | 大硅片/光刻胶突破中 |
| **EDA** | ~15%（目标2030年40%） | 华大九天、概伦电子 | 存储全流程已突破 |

### 一、政策与资金

#### 1.1 大基金三期

- **规模**：2024年5月成立，注册资本**3440亿元**，超过一期（987亿）+二期（2042亿）之和
- **股东**：财政部、国开金融、四大行+交行等19家
- **投资方向**：明确"设备材料国产化（70%资金）+先进封装与AI存储（30%资金）"双主线
- **已出手规模**：截至2025年底已投资**1640亿元**，设立国家集新、华芯鼎新两只基金
- **典型标的**：拓荆科技子公司拓荆键科获增资4.5亿元（晶圆键合设备，HBM封装核心）

#### 1.2 十五五规划（2026-2030）

- 集成电路列为**战略性新兴支柱产业之首**
- 提出"采取**超常规措施**，全链条推动关键核心技术攻关取得**决定性突破**"
- 完善"揭榜挂帅""赛马"制度，探索以奖代补、后补助等资金支持
- 目标：2027年半导体设备国产化率突破70%；2030年EDA国产化率40%+

#### 1.3 科创板支持

- 2025年6月设置**科创成长层**，重启未盈利企业上市通道（第五套标准）
- 聚焦"技术突破大、商业前景广、研发投入高但尚未盈利"的科技企业
- 2025年壁仞科技（港股）、摩尔线程、沐曦股份先后上市，百度昆仑芯启动独立IPO

### 三、卡脖子环节排名（由难到易）

#### 3.1 极难突破（国产化率<5%）

1. **EUV光刻机**：ASML垄断（全球市占82%），中国无法获取。上海微电子最新仅达DUV 28nm级别，EUV差距约10年。国产光刻机整体国产化率仅1.1%
2. **先进制程（5nm以下）**：受限于EUV缺失，中芯N+2（~7nm）已是极限，多重曝光（multi-patterning）勉强实现但良率65%远低于台积电85%，成本高出30%+
3. **涂胶显影设备**：国产化率仅1.0%，东京电子（TEL）垄断
4. **高端GPU/AI训练芯片**：华为昇腾910C FP16算力仅为H100的~19%（~380 vs 1979 TFLOPS），千卡集群训练效率不足30%（H100达90%+），NVLink互联带宽差距4.5倍

#### 3.2 较难突破（国产化率3-10%）

5. **过程控制/量测设备**：3.6%，KLA垄断
6. **离子注入设备**：3.1%，应用材料垄断
7. **EDA全流程**：仅覆盖~15%市场，高端数字后端/签核工具差距大
8. **ArF/EUV光刻胶**：ArF仅2%，EUV在研

#### 3.3 加速突破中（国产化率10-40%）

9. **薄膜沉积设备**：5.7%→快速提升，拓荆/北方华创
10. **刻蚀设备**：22%，中微/北方华创已突破14nm
11. **12英寸大硅片**：沪硅产业月产60万片，国产化率~30%
12. **CMP材料/设备**：23%，安集科技+华海清科

#### 3.4 基本可控（国产化率>40%）

13. **封测**：60%+，长电/通富/华天全球前十
14. **清洗设备**：38%，盛美上海
15. **去胶设备**：74%，屹唐/盛美
16. **成熟制程制造（28nm+）**：80%+设备覆盖率

### 四、AI芯片国产化深度分析

#### 4.1 竞争力对比

| 指标 | 华为昇腾910C | 寒武纪MLU590 | 海光深算三号 | NVIDIA H100 | NVIDIA H20（对华版） |
|------|-------------|-------------|-------------|-------------|-------------------|
| 制程 | 7nm | 7nm | 7nm | **4nm** | 12nm |
| FP16算力 | ~380 TFLOPS | ~256 TFLOPS | ~200 TFLOPS | **1979 TFLOPS** | 148 TFLOPS |
| 显存 | 80GB HBM3 | 64GB HBM2e | HBM2e | 80GB HBM3 | 96GB HBM3 |
| 带宽 | 3.35 TB/s | 1.6 TB/s | 1.6 TB/s | **4.8 TB/s** | 4.0 TB/s |
| 互联 | 200 GB/s | 有限 | 有限 | **900 GB/s NVLink** | 450 GB/s |
| 软件生态 | CANN（适配中） | Cambricon Neuware | ROCm兼容 | **CUDA（400+库）** | CUDA |
| 实际利用率 | 58% | ~50% | ~55% | **92%** | 85%+ |

#### 4.2 差距本质

1. **制程差距**（4nm vs 7nm）：晶体管密度差1.8倍，直接影响算力密度
2. **互联瓶颈**：NVLink 900GB/s vs 昇腾200GB/s，千卡集群效率差距3倍+
3. **软件生态鸿沟**：CUDA生态400+专用库 vs 国产配套不足1/3，适配周期数月
4. **HBM供应**：SK海力士/三星垄断HBM3e，国内仅长鑫存储在研HBM2e

#### 4.3 发展趋势

- 华为Atlas 350推出（2026.3）：单卡算力=2.87x H20，推理场景接近实用
- 寒武纪2025营收暴增453%至65亿，市占率超35%
- Bernstein预测：2026年华为国内AI加速器市占率提升至**50%**，NVIDIA因禁售降至8%
- DeepSeek等国产大模型对推理效率优化，部分弥补硬件差距
- **核心判断**：训练场景差距仍大（2-3代），推理场景2026年可基本满足需求

### 五、投资机会与风险

#### 5.1 确定性最高的细分赛道

1. **半导体设备**（确定性最高）：大基金三期70%资金投入，设备国产化从50%→70%空间明确。北方华创/中微/拓荆/盛美均保持30-55%增速
2. **AI芯片/算力**：国产AI芯片需求刚性（禁售倒逼），寒武纪/海光信息订单饱满
3. **先进封装**：HBM/Chiplet封装需求爆发，长电科技/通富微电直接受益
4. **存储芯片**：HBM产能迁移腾出利基市场空间，兆易创新/江波龙受益

1. **政策风险**：美国进一步升级制裁（MATCH法案扩大至DUV光刻机），或放松制裁（减弱国产替代紧迫性）
2. **技术瓶颈**：EUV光刻机短期（5-10年）难以突破，先进制程天花板明确
3. **产能过剩**：成熟制程（28nm+）大量扩产，已出现价格战苗头
4. **估值泡沫**：板块PE处历史97%分位，一旦业绩不达预期，回调风险大

7. **周期风险**：半导体行业强周期属性，2025-2026景气高点后可能回落
