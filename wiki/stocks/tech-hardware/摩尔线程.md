## 投资动作建议

- 当前 ~717 元 = 距合理市值中位 1500 亿（~320 元）约 **+124%**（极端高估） — **不建议入场**
- **350-450（关注价）小幅试探 / 180-230（买入价）做底仓** — 需 -55% 才到关注价
- **核心观察时间表**：
  - **2026-08 中报**：H1 营收能否维持 +150%+ 增速 + 现金流是否持续转正 + 万卡集群项目交付节奏
  - **2026Q3 MTT S5000 大规模出货**：CUDA 兼容性能损耗能否控制在 < 20%（关键技术指标）
  - **2027 全年盈亏平衡**：管理层指引 2027 实现年度盈利能否兑现
  - **解禁压力**：原始股东（含一级市场大额融资方）解禁节点（**2026-12 / 2027-12 关注**）
- **触发上调情景**（任三达成则合理 PS 50x → 70x，合理市值 2100 亿）：H1 营收 ≥ 20 亿（+200% 增速）+ 万卡集群签约 ≥ 5 个 + MTT S5000 在头部互联网客户落地 + NV H20 / Blackwell 对华出口进一步收紧
- **触发下调情景**（任一发生则合理 PS 50x → 30x，合理市值 900 亿）：央企万卡项目延期/收缩 + 互联网客户优先选择华为/寒武纪 / CUDA 兼容性能损耗 > 30% 暴露 / 解禁后大股东减持 + 营收增速 Q2 起跌破 100%
- **不加入 focus list**（v1 暂观察，估值过高 + 财务尚未稳态，待 2026 中报 + Q3 S5000 出货数据后再评估）

## 红灯信号

1. **央企万卡集群项目延期**：中国移动 / 中国联通 / 央能任一项目延期 6 个月以上，叙事崩塌
2. **CUDA 兼容性能损耗暴露**：若实测兼容 CUDA 时性能损耗 > 30%，"NV 路线降迁移门槛"叙事破产
3. **大股东解禁后减持**：摩尔线程一级市场融资轮估值 298 亿（2024 末），当前 3370 亿是 11x 浮盈，**解禁后大额减持概率极高**
4. **新基金 IPO 退潮**：与沐曦股份联动，若新股稀缺性溢价随后续 IPO 增加（百度昆仑芯、燧原等）而消退

## 企业类型与估值模型

### 类型判定

**早期亏损 GPU 公司**（与地平线机器人同框架，但**走 NV 全功能 GPU 路线**而非 ASIC）：
- 营收**单边爆发但基数小**：2022 < 0.5 亿 → 2023 1 亿+ → 2024 4.38 亿 → **2025 ~15 亿（+242%）** → 2026Q1 7.38 亿（+155%）
- 净利**仍为负**：2022-2024 亏损 18.4/16.73/14.92 亿；2025 预亏 ~10 亿；**2026Q1 首次单季盈利 0.29 亿**
- 客户**B2B + 央企导向**：自上而下的政企/运营商路径，不同于寒武纪的互联网路径

### 估值方法适用度

| 方法 | 适用度 | 角色 | 备注 |
|---|---|---|---|
| **P/S** | ★★★★★ | **主** | 营收高增 + 仍亏损 |

- **中性合理市值 = 50x × 30 亿 = 1500 亿**
- 当前 3370 亿 / 1500 亿 = **2.25x（明显高估）**

### 模型二：DCF 辅估值（权重 30%）

#### 假设链（[估计]）

- **2026-2030 收入复合增速**：60%（前 2 年 +100%/+70%，后 3 年 +40%/+30%/+20%）
- **稳态净利率**：25%（GPU 通用计算 + 图形渲染双业务，给国产折让；低于寒武纪 35%）
- **永续增速 g**：3%
- **折现率 WACC**：14%（早期亏损 + 解禁压力，WACC 高于寒武纪）
- **2030 收入**：15 × 1.6^5 ≈ 157 亿 → 2030 净利 ≈ 39 亿
- **2030 终值**：39 × (1+3%) / (14% - 3%) ≈ 366 亿
- **DCF 现值合计**：约 **1000-1300 亿**（[估计 ±25%]）

**DCF 锚**：市值 **1150 亿**（中位）

## 业务全景

### 产品矩阵

**MUSA 架构（自研指令集 + 工具链）**：
- 第一代（MUSA 1.0）：MTT S60 / S80（消费级图形 + AI 入门）
- 第二代（MUSA 2.0）：MTT S2000 / S3000 / S4000（数据中心入门）
- **第三代（MUSA 3.0）**：**MTT S5000**（曲苑芯片）—— 当前主力出货 SKU

**MTT S5000 关键参数** [观测 摩尔线程官网]：
- 峰值算力 **1000 TFLOPS**（精度未明，估 FP16 [估计]）
- 显存 **80GB**（HBM）
- 显存带宽 **1.6 TB/s**
- 卡间互联 **784 GB/s**
- 支持 FP8 到 FP64 全精度
- 训推一体 + 图形渲染 + 视频编解码 + 8K HDR 显示

**KUAE（夸娥）智算集群**：
- 从千卡级扩展至万卡级
- 支持万亿参数大模型训练
- 2025-12 与中国移动、中国联通青海、央能等签约 3 个万卡集群项目

### 客户结构（B2B + 央企导向）

- **中国移动**：MTT S5000 适配九天 35B 大模型，深度生态合作伙伴
- **中国联通青海**：万卡集群项目
- **中国能源建设、桂林华觉大数据**：智算中心建设项目
- **联合发布 OISA 高密度超节点设计规范**（与中国移动研究院）

**对比寒武纪**：寒武纪走"互联网巨头"路线（字节、阿里、腾讯），客单价高但集中度高；摩尔线程走"央企/运营商"路线，客户更分散但单价偏低、回款周期长。

### 技术路线 vs 竞品

| 项目 | 摩尔线程 MTT S5000 | 寒武纪 思元 590 | 华为 昇腾 910C | NVIDIA H100 |
|---|---|---|---|---|
| 制程 | 7nm [估计] | 7nm | 7nm（中芯 N+2） | 4nm |
| 算力（FP16）| 1000 TFLOPS [估计 = 峰值] | ~400 TFLOPS [估计] | ~320 TFLOPS | 989 TFLOPS |
| 显存 | 80GB HBM | 64GB HBM | 128GB HBM | 80GB HBM3 |
| 带宽 | 1.6 TB/s | ~1.6 TB/s | 3.2 TB/s | 3.35 TB/s |
| 软件栈 | **MUSA + 兼容 CUDA** | Cambricon Neuware（自研） | CANN（自研） | CUDA（全球生态） |
| 业务范围 | **图形 + AI + HPC** | 仅 AI | 仅 AI | 全功能 |
| 生态 | 中等（兼容 CUDA 是关键卖点） | 中等（自研） | 中等（自研） | 顶级 |

**核心矛盾**：
1. **CUDA 兼容性能损耗**：摩尔线程兼容 CUDA 是关键卖点（降低客户迁移门槛），但兼容层性能损耗未公开，是关键技术不确定性
2. **算力 1000 TFLOPS 含金量**：参数纸面与 H100 接近，但实际训练效率受互联带宽（784 GB/s vs H100 900 GB/s NVLink）+ 软件成熟度限制，**实测往往 50-70% [估计]**

### 成长性催化与风险

**催化（看多）**：
1. **NV 路线唯一性**：A 股唯一走 NVIDIA 全功能 GPU 路线 + 兼容 CUDA
2. **央企万卡集群放量**：三个项目签约 + OISA 规范联合发布
3. **图形渲染 + AI 双业务**：相比寒武纪只做 AI，业务多元化
4. **国产替代政策加码**：信创扩围 + 党政机关 GPU 替换

**风险（看空）**：
1. **估值极端**：PS 225x，全球 GPU 同业最高水平
2. **解禁压力**：2026-12 / 2027-12 大股东解禁
3. **CUDA 兼容性能损耗**：实测数据未公开
4. **互联网客户冷淡**：沐曦自承"互联网企业首选国际产品"，摩尔线程客户路径类似
5. **行业内同质化**：百度昆仑芯独立 IPO 推进、燧原、壁仞陆续上市，稀缺性溢价将稀释
6. **存货 / 应收账款风险**：央企项目回款周期长
