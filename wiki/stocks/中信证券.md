## 基本信息
- 市场：A股（上证）+ 港股（6030.HK）
- 行业：证券 / 金融
- 市值：约 3839 亿元（2026-04-10）

## 投资逻辑
- 中国券商行业绝对龙头，营收和净利长期行业第一
- 全牌照业务布局：经纪、投行、资管、自营、信用全面领先
- 受益于资本市场改革：注册制、衍生品扩容、财富管理转型
- 合并国泰君安预期（如落地将进一步巩固龙头地位）

### 关键趋势
- **强周期特征**：营收和利润随市场行情大幅波动，2021年牛市净利231亿，2023年弱市降至197亿
- **2025年强劲反弹**：前三季度营收+32.7%、净利+37.9%，受益于A股成交量放大和资本市场回暖
- **ROE偏低**：长期在8-12%区间，受高杠杆和低ROA制约，不如消费品公司
- **现金流极不稳定**：2023年-408亿，2024年+1726亿，与自营投资和融资业务波动直接相关
- **净利率持续提升**：从28.5%提升至35.4%，经营效率改善
- **分红逐步提升**：2024年两次分红合计约0.57元，股息率约2.2%

## 风险点
- 强周期性：业绩高度依赖市场行情，熊市利润可腰斩
- 政策风险：监管收紧（如IPO节奏调控、降佣金）直接冲击收入
- 资产质量不透明：自营和信用业务的风险敞口外部难以评估
- ROE天花板：券商商业模式决定ROE难以持续超过12%
- 同质化竞争：头部券商业务差异化不足

### 合理PE推导

**因子模型 → 14x：**

| 项目 | PE | 说明 |
|------|-----|------|
| 行业基础PE | 12x | 金融（券商） |
| + 行业龙头 | +3x | 营收净利均为行业第一，领先第二名70%+ |
| + 净利率改善 | +1x | 净利率从28.5%提升至35.4% |
| + 分红提升 | +1x | 开始实施半年度分红，股东回报意识增强 |
| - 强周期波动 | -2x | 2021净利231亿→2023仅197亿，波动幅度大 |
| - 资产质量不透明 | -1x | 自营和信用业务风险敞口外部难以评估 |
| - ROE偏低 | 0x | ROE 8%对券商属正常，不额外减分 |

**市场定价校验：**
- 近半年PE中枢：15.0x（均价27.5元 / TTM EPS 1.83）
- 模型14x vs 市场15x，偏差+7% → **无显著分歧**
- 不修正，但需注意：2025年利润大幅增长(+38%)推高了TTM EPS，若市场转冷EPS可能回落
- 合理PE保持 **13x**（对TTM EPS打折：券商EPS在牛市年份偏高，用13x而非14x更保守）

确定性等级：低确定性 → 基础安全边际35%（券商强周期，盈利不确定性大）

- 当前PE(TTM) 11.7x | 合理PE 13x | 比值 90% → **预期偏低**
- 增速趋势：2025全年+38.6%，2026Q1继续+54.6%，景气上行期
- 但注意：高增速有低基数效应+市场行情驱动，券商利润随行情波动大，不可线性外推
- 安全边际调整：不调整

### 关键关注点

1. **市场成交量**：日均成交额是券商利润的领先指标，若持续>1万亿则利润有支撑
2. **与国泰君安合并进展**：若落地将产生规模协同效应和市值重估
3. **PB 1.0x（约18.9元）是极端安全边际**：历史上PB跌破1.0x的机会极少，出现即是重大买入机会
