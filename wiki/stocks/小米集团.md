## EPS基线说明

小米报表利润口径复杂，三层差异显著：

```
GAAP归母净利润     416亿  ← 含大额投资公允价值变动收益
 减：投资FV变动/SBC等   -24亿
= 经调整净利润      392亿  ← 市场共识口径（加回SBC，剔除部分投资损益）
 减：投资收益等     ~-139亿
= 核心经营净利润    ~253亿  ← 纯主业经营利润

→ 估值采用：经调整EPS = 392亿 / 259亿股 × 1.086汇率 = 1.64 HKD
→ 当前调整后PE = 30.60 / 1.64 = 18.7x
```

注意：调整后PE 18.7x看似合理，但30%+利润来自投资组合（430家被投企业），核心PE实际约29x。

## 风险提示

**高风险：**
- **DRAM成本飙升**：存储芯片涨价4倍，直接冲击手机+IoT毛利率。如果2026全年维持高位，手机毛利可能从203亿降至138亿（-32%），侵蚀整体利润。DRAM是周期品，但本轮受AI需求驱动，持续时间不确定。
- **手机销量下滑**：内部预计2026年出货量-13%，高端化战略能否对冲ASP下降是关键。
- **AI投入回报不确定**：2026年AI领域投入≥600亿，包括自研大模型、AI芯片、机器人。短期压缩利润，长期回报不确定。

**中等风险：**
- **EV产能爬坡**：55万辆目标激进（Q1仅交付7.9万辆），需月均5.2万辆，任何产能/交付问题都会影响全年目标。
- **投资组合波动**：430家被投企业FV变动大，2025年贡献了~163亿利润，但在市场下跌时可能反向。
- **关税/地缘风险**：海外市场占比大，中美贸易摩擦可能影响供应链和海外销售。

## 投资逻辑

**看多理由：**
- 股价腰斩至30港元，PE 18.7x为近4年较低水平（历史低点PE 15.9x）
- EV业务"从0到1"已完成（首年盈利），"从1到N"正在加速（2026目标55万辆）
- AIoT生态（手机+车+家电+穿戴）竞争壁垒持续加深
- DRAM是周期品，2026H2-2027可能见顶回落，手机利润有望V型反转
- 互联网服务76.5%毛利率的现金牛业务被低估

**看空理由：**
- 2026年是"失速年"：手机利润坍塌+AI烧钱，调整后净利增速从44%→5%
- 核心经营净利仅253亿（不含投资收益），核心PE高达29x
- EV竞争白热化（比亚迪427万辆 vs 小米41万辆），规模差距10倍
- 600亿AI投入短期无法变现，可能成为"投入黑洞"
