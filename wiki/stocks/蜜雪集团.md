## 核心投资逻辑

蜜雪冰城的本质不是一家奶茶公司，而是一家 **to B 供应链公司**。它赚的是卖原材料和设备给加盟商的钱，而非直接面向消费者。这决定了它的商业模式更接近于"卖铲子的人"。

**看多理由：**
- 全球门店近6万家，现制饮品全球第一（超越星巴克），规模效应极强
- 供应链护城河深：核心原料100%自主生产，28个国内仓库+8国海外仓储，成本压到行业最低
- 财务质量极高：ROE ~30%、净利率17.5%、几乎零负债（负债仅1.87亿）、现金163.6亿
- 2025年报超预期：营收+35.2%、净利+32.7%，增速远超同行
- 估值回落至合理区间：PE从上市高点~35x回落至15.4x，Forward PE仅12.8x
- 下沉市场+海外双引擎：国内下沉空间仍在，海外已进入14国

**看空/风险理由：**
- 股价距上市高点腰斩（618→276港元），市场担忧增速放缓
- 2026年主动放缓国内开店，增长逻辑从"门店扩张"切换到"单店提质"，市场接受度存疑
- 单店营收数据未披露，外卖补贴退潮可能冲击加盟商单店收入
- 海外扩张不及预期，部分市场已关店调整
- 新茶饮行业整体竞争激烈，茶百道/古茗/奈雪等均承压

## 商业模式与护城河

### 供应链壁垒（核心护城河）

```
上游自建生产基地 → 核心原料100%自产 → 28个国内仓库 + 8国海外仓储
    → 配送覆盖33省300+地级市 → 成本行业最低 → 加盟商拿货价低
        → 终端售价2-8元仍有利润 → 门店扩张 → 规模反哺供应链
```

这是一个 **正循环飞轮**：门店越多 → 采购规模越大 → 成本越低 → 加盟商越赚钱 → 吸引更多加盟商 → 门店越多。

## 2026年战略转型

管理层在年报中明确：
1. **国内放缓开店**：聚焦存量门店运营扶持与效益提升
2. **蜜雪冰城**：围绕"真鲜纯"升级产品，高频上新+布局旗舰店
3. **幸运咖**：已破万店，2026年"提质增效"，开店速度放缓
4. **海外**：14国4,467家，继续拓展但更注重单店模型跑通
5. **CFO接棒CEO**：管理层换帅，从创业者驱动转向职业经理人治理

## 风险点

1. **增速放缓预期**：2026E收入增速从35%降至15%，市场可能给予更低估值倍数
2. **单店营收不透明**：公司未披露单店数据，市场担忧门店饱和+外卖补贴退潮冲击加盟商收入
3. **海外扩张不确定性**：部分市场已关店，不同国家的供应链+文化适应成本高
4. **食品安全**：加盟模式下品控难度大，一旦出现食安事件对品牌冲击巨大
5. **新茶饮行业内卷**：同行降价、产品同质化，可能拉低整体行业利润率
6. **股价与IPO预期差**：上市首日涨43%后持续下跌，早期投资者套牢严重

**总结：** 蜜雪集团是少有的 **高ROE+高增长+低杠杆+强护城河** 的消费标的。估值模型给出合理价值432港元，当前275.2港元显著低于安全边际买入价345.6港元（低20.4%），处于明确的**买入区间**。市场从IPO高点618跌至275的过程中，已将增速放缓+海外不确定性充分Price in（PE从34x杀至15x），但基本面（ROE 30%+、净利+32.7%、零负债）并未恶化。PE+DCF+远期三模型交叉验证一致指向400-450 HKD的合理价值区间，上行空间约57%。Forward PE 12.7x对应13%的利润增速，PEG约1.0x。

**核心风险：** 若2026年增速低于10%（即低于保守假设），合理PE需下修至20-22x，合理价值将降至360-395 HKD，但仍高于当前价30%+。
