# 价值陷阱排雷框架

> 低估值投资在 A 股最大的风险是踩进价值陷阱。本框架整合两层财务排雷工具（**显性低估陷阱** + **商业模式/现金流陷阱**），在不依赖深度行业认知的前提下，用基础财务指标排除约 70-80% 的常见陷阱。第三层（认知型陷阱）将在后续补充。

## 何时用 / 不用这个框架

**用：**
- 看到一只 PE/PB 显著低于行业均值的票，下买入决策前
- 高分红股票要判断分红可持续性
- 周期股估值锚定时，区分"周期底部" vs "结构性衰退"
- 把候选标的放进 focus list 之前的初筛

**不用：**
- 银行/保险/证券（现金流框架失效，见"框架边界"）
- 再投资型成长股（FCF 为负不等于陷阱，见"框架边界"）
- 已确认的高质量复利机器（如茅台）反复体检，浪费精力

## ⚠️ 框架边界

| 不适用的企业类型 | 为什么 | 替代框架 |
|---|---|---|
| 银行 | "经营现金流"含存款贷款变化，与净利润无对标关系 | ROA × 杠杆 / 拨备前利润 / 不良生成率 / 净息差 |
| 保险 | 利润表是会计驱动结果，与现金流脱节 | 内含价值（EV）/ 新业务价值（NBV）/ 综合成本率 |
| 证券 | 收入波动大、自营投资影响大 | ROE 长期均值 / 业务结构 / 风控指标 |
| 再投资型成长股（比亚迪/宁德/长城/亚马逊式）| 高资本开支期 FCF 长期为负，按本框架会误判 | 看 ROIC 是否持续 > WACC，P/S 主估值 |
| 公用事业型（中移动/电力/高速）| ROE 低于框架门槛但稳定 | DDM / 股息增长率 / 长期 FCF 稳定性 |

**走完此框架不构成陷阱 ≠ 值得买**。本框架只解决"排除"问题，不解决"选择"问题。

## 第一层：显性低估陷阱（看估值数字）

### 三类陷阱画像

**1. 低 PB 陷阱**

- 企业市值低于净资产，但靠清算获利在 A 股几乎无效（国资控股 + 行业快速迭代使烟蒂股迅速贬值）
- 典型反例：地产股、部分基建股 PB < 0.5x 但 ROE 持续走低、资产质量恶化

**2. 低 PE 陷阱**

| 子类型 | 画像 | 反例 |
|---|---|---|
| 周期顶部低 PE | 周期股盈利顶部时 PE 最低，但估值实际最高（山顶接盘）| 稀土、面板、煤炭、海运 |
| 衰退企业低 PE | 盈利能力持续下滑，市场给折价 | 部分传统媒体、传统零售 |

**3. 双低（低 PE + 低 PB）陷阱**

| 子类型 | 画像 | 反例 |
|---|---|---|
| 高杠杆陷阱 | 资产负债率 60%+，普通投资者看不懂资产质量 | 部分地产、部分股份行 |
| 双低 + 低 ROE | PE × ROE = PB，市场公允定价，估值难回归 | 部分建筑央企（PB 0.3 / ROE 5%）|

### 第一层四步排雷

| 步骤 | 检查项 | 触发动作 |
|---|---|---|
| 1 | 仅 PB 低、利润糟糕？ | 直接 pass |
| 2 | 跨周期 10 年财务数据 | 排除周期顶部低 PE 陷阱；提示连续 10 年造假概率虽低但不为零（康美/康得新案例提醒）|
| 3 | 长期 ROE vs PE 自洽性 | PE × ROE ≈ PB，若三者自洽则估值是公允定价不是低估 |
| 4 | 资产负债率 > 60%？ | 高杠杆行业要么深入理解（招行/平银可投，需读懂资产质量），要么避开 |

### 第一层警示

**"长期合理 PE ≈ 长期 ROE"是一种简化口诀，准确关系是 P/B ≈ ROE / r（戈登模型简化）**。茅台 ROE 30%+ 不代表合理 PE 就是 30 倍——折现率 r 也很关键。

## 第二层：商业模式 + 现金流陷阱（看赚钱质量）

### 四个核心问题

**对应 DCF 的拆解**：
- 问题 1+2 = 当前自由现金流（FCF₀）真不真
- 问题 3 = FCF 持续性（贴现率 r）
- 问题 4 = FCF 增长率（g）

#### 问题 1：能不能赚钱（双高指标）

| 指标 | 底线门槛 | 优质门槛 | 注意 |
|---|---|---|---|
| 长期稳定 ROE | > 12% | > 15% | 必须长期均值，不是单年峰值 |
| 销售毛利率 | > 15% | > 20% | 零售/分销/物流天然低毛利不适用 |

**正面信号**：ROE + 毛利率持续上升 → 赚钱能力提升
**负面信号**：ROE + 毛利率持续下滑 → 商业模式恶化

**门槛失效场景**：高周转低毛利商业模式（沃尔玛/Costco 毛利 13% 但 ROE 25%）、公用事业型（中移动 ROE 9% 但稳定）。**修正版**："**ROE × FCF 转换率 > 阈值**"才是更准确的能赚钱判据。

#### 问题 2：赚的是不是真钱

**核心指标：**

| 指标 | 公式 | 健康区间 |
|---|---|---|
| 利润现金流含量 | 经营净现金流 / 净利润 | 长期 ≈ 1.0 |
| 自由现金流（FCF） | 经营净现金流 - 资本开支 | 长期 > 0 |
| 分红自由现金流支撑度 | 分红总额 / 自由现金流 | < 1.0（< 1 才可持续）|

**典型陷阱画像：**
- 利润表"看着不错"，但 OCF 长期远低于净利润 → 利润表是会计魔术
- 高分红 + 分红 / FCF > 1 → 分红是借钱发的，不可持续
- OCF > 净利润但 FCF 持续为负 → 钱赚到了又被砸进资本开支（除非是再投资型成长股，看问题 4）

**白酒高分红陷阱反例（文章作者举例）**：某白酒企业 2020-2024 年 OCF/净利润比值持续偏低，分红现金流出 > OCF 现金流入，是典型高分红陷阱。**注意**：本 wiki 的"五粮液财务大洗澡事件"（2026-04-30）也属于这一类的极端版本——通过会计操纵掩盖渠道压货导致的"利润不是真钱"。

#### 问题 3：能不能持续赚钱（行业格局 + 资产护城河）

**横向对比 TOP3 企业**（市值 / 销毛 / ROE）：
- 三者持续走平/上升的是龙头，下滑的是失败者
- 例：乳业 - 伊利 ROE 20% / 蒙牛光明持续下滑 → 伊利在持续赢

**资产护城河分级：**

| 等级 | 类型 | 特征 | 例 |
|---|---|---|---|
| 上等 | 无形资产 | 弱竞争 + 自我增值 | 品牌、平台生态、网络效应 |
| 中等 | 半无形资产 | 弱竞争 + 相对保值 | 专利、专营权、有区位优势的固定资产、公用事业 |
| 下等 | 有形资产 | 强竞争 + 快速折旧 | 技术、设备、建筑、存货 |

**注意**：分级是粗线条，**同一企业不同业务的资产类型可能跨级**（如腾讯：微信生态=上等 / 游戏=技术驱动偏中下等）。

#### 问题 4：能不能赚越来越多的钱（再投资效率）

**核心是再投资效率（ROIIC），其次才是市场空间**：
- 即使空间大，再投资低效（如光伏价格战）→ 价值毁灭
- 即使空间看似有限，竞争优势强 → 总能找到出路（如茅台扩产受批文限制反而保护稀缺性）

**反例画像（光伏型恶性竞争）**：
- 出货量高速增长（量增）
- 但收入增速放缓 / 利润下滑（量价不齐）
- 销毛 + ROE 同步下滑（资产回报率恶化）
- 持续大额资本开支（必须不断技术迭代）
- → 头部企业也无定价权

**判据**：FCF 为负的成长股必须看 **ROIC 是否持续 > 10%**（资金成本）。是 → 再投资创造价值（比亚迪/宁德）；否 → 烧钱无回报（多数光伏/储能/创新药 biotech）。

## 整合排雷流程

```
┌─ Step 1: 显性估值检查（第一层）────────────────────────┐
│  - 只 PB 低？ → pass                                  │
│  - 周期顶部低 PE？ → pass                             │
│  - 高杠杆看不懂？ → pass                              │
│  - PE × ROE ≈ PB 自洽？ → 公允定价不是低估            │
└────────────────────────────────────────────────────┘
                          ↓ 通过
┌─ Step 2: 框架适用性判断 ──────────────────────────────┐
│  - 是银行/保险/证券？ → 切换到金融专用指标             │
│  - 是再投资型成长股？ → ROIC 体检 + P/S 估值          │
│  - 是公用事业型？ → DDM + 股息稳定性                  │
└────────────────────────────────────────────────────┘
                          ↓ 适用
┌─ Step 3: 四问体检（第二层）───────────────────────────┐
│  Q1 双高（ROE > 12%, 毛利 > 15%）？                   │
│  Q2 OCF/净利润 ≈ 1 + 分红 / FCF < 1？                 │
│  Q3 行业 TOP3 横向对比，本公司在赢？                  │
│  Q4 ROIC 持续 > 10% + 再投资创造价值？                │
└────────────────────────────────────────────────────┘
                          ↓ 全过
              候选标的进入估值流程
              （走『估值方法选择框架』）
```

**任何一步触发 pass，都不再进入下一步**。
