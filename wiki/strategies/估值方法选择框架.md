# 估值方法选择框架

> 分析任何标的之前先做这一步——基于公司行业 + 营收/销量/净利率历史特征，
> 判断主估值方法，避免"先开干再发现方法选错"的迭代浪费。

## ⚠️ 框架边界（避免虚假安全感）

**本框架只解决"方法选择"问题（约占踩坑场景 5%）**。剩余 95% 的踩坑必须靠分析师常识与单独流程：

- **数据真实性**（财务造假/ROE 失真，如康美/康得新）—— 再好的估值方法也救不了
- **一次性会计调整**（如中移动 Q1 增值税教训）—— 这是 EPS 基线问题，不是方法选择
- **永续假设崩塌**（如白酒人口论）—— 这是 g 假设问题
- **预期差/情绪定价**（短期市场波动 ±20%）—— 估值方法管不到

**走完此框架 ≠ 估值正确**。它只是排除了"用错工具"这一种错误。保留怀疑态度。

## 为什么需要这一步

长城汽车 2026-04-28 三轮迭代教训：
- v1 用 TTM PE → 14 元
- v2 用中周期 PE → 20-21 元
- v3 用 P/S 主 → 24 元（与市场实际定价 -16% 折价吻合）

**根因**：把"利润率周期波动"当成"周期股"，错用 PE/PB。三次重写的成本可以通过**前置 5 分钟方法判断**完全避免。

## 🌳 兜底入口：决策树（新子行业用）

依次回答 5 个问题（**关键规则：Q2 通过即跳过 Q3**）：

1. **行业是否金融**（银行/保险/券商/信托）？→ 是 → **PB 主**（PE 辅）
2. **营收是否单边上行**（5-10 年仅 1-2 年微跌 < 5%，无大幅萎缩）？
   - **是** → 跳到 Q4（**不再问 Q3**——工业品营收受价格×量双驱动，营收单边已隐含价格能消化销量波动）
   - **否** → 进入 Q3
3. **销量/产量是否同步周期波动**（萎缩年明显，跟随经济周期）？→ 是 → **真周期股，PB 主**
4. **净利率是否长期稳态**（5 年波动幅度 ≤±20%，**先剔除单年极端值**）？→ 是 → **稳态消费/成长，PE 主**
5. **以上都否**（营收单边 + 净利率周期波动 ±50%+）→ **弱周期成长，P/S 主**

### 阈值口径定义（避免不同分析者得出不同结论）

**净利率波动幅度** = `(max - min) / 中位数 × 100%`

**计算前先剔除单年极端值**（偏离 5 年中位数 > 2 倍标准差的年份）。

> 避免高净利率公司（医药/科技）单年暴跌被错归"弱周期成长"。
> 例：某公司 5 年净利率 25% / 26% / 27% / 8% / 26%
> - 不剔除：(27-8)/26 = 73% → 错归"弱周期成长"
> - 剔除单年异常 8%：(27-25)/26 = 8% → 归"稳态消费/成长" ✓

示例（长城汽车 5 年净利率 4.0% / 4.9% / 6.0% / 6.3% / 4.4%，无单年极端值）：
- 中位数 4.9%、max 6.3%、min 4.0%
- 波动 = (6.3 - 4.0) / 4.9 ≈ **47%** → 落在 ±20%~±50% 之间，归"弱周期成长"
- 验证：实际 v3 即采用此分类，与决策结果一致

### 混合业务公司的特殊规则

**主业占比 < 70% → 必须用 SOTP 分部估值**（不再走单一主估值）。

- 例：腾讯（游戏 ~30% + 广告 ~20% + 金融科技 ~30% + 云 ~10% 等）→ SOTP 主
- 例：阿里（电商 ~60% + 云 ~12% + AIDC ~8% 等）→ 主业接近 70% 临界，混合 SOTP

**单一主业占比 ≥ 70%** → 走决策树/快速通道选主估值。

### Step 0 vs Step 4.1 分类的关系

> ⚠️ 本框架是 **5 类粗分类**（金融/真周期/稳态消费/弱周期成长/高成长），用于 value-invest skill 的 **Step 0 主估值方法选择**。
> Skill 的 **Step 4.1** 是 **7 类细分类**，多出的"利润改善型/高分红型/困境反转型"**是 PE 主估值下的细分**，不是本框架的并列项。
> **Step 4.1 在 Step 0 判完类后仅作查表确认**，不再独立判类（避免与本框架矛盾）。
> 例：转型期青啤本框架落到"稳态消费 PE 主"（Q4 通过），Step 4.1 进一步细分为"利润改善型"以微调因子。

## 五类企业 × 估值方法对照

### 1. 稳态消费 / 成长（PE 主）

| 维度 | 表现 |
|------|------|
| 识别 | 量价稳态、ROE 长期 >15%、净利率波动 ≤±20% |
| 主估值 | **PE**（合理 PE = 行业基础 + 因子加减）|
| 辅 | DCF（现金流可预测时）、SOTP（多业务线）|
| 不用 | PB（资产周转不是核心）、PEG（增速稳定）|
| **个股案例** | 贵州茅台 / 青岛啤酒 / 蜜雪集团 / 腾讯 / 美团 / 药明 |

### 2. 金融（PB 主）

| 维度 | 表现 |
|------|------|
| 识别 | 银行/保险/券商，杠杆经营 |
| 主估值 | **PB**（合理 PB = 正当 ROE / 要求回报率）|
| 辅 | PE（盈利稳定时）、DDM（高股息银行）|
| 不用 | DCF、PEG |
| **个股案例** | 平安银行（PB 主）|

### 3. 真周期（PB 主）

| 维度 | 表现 |
|------|------|
| 识别 | **营收+销量同步周期波动**（钢铁/煤炭/水泥/有色），萎缩年常见 |
| 主估值 | **PB**（历史 PB 区间定价）|
| 辅 | PE（仅周期底参考）、商品价格×产量 |
| 不用 | PEG、DCF（FCF 波动）|
| **个股案例** | 海螺水泥（PB 主）/ 紫金矿业（PB+PE 双主）|

### 4. 弱周期成长（P/S 主）★

| 维度 | 表现 |
|------|------|
| 识别 | **营收单边上行 + 利润率周期波动**（2-7% 区间，受价格战/原材料/汇率扰动）|
| 主估值 | **P/S（市销率）**（基准 = 全球可比龙头，调整 = 规模/成长/海外/净利率）|
| 辅 | 中周期 PE（营收×正常化净利率×PE）、SOTP（多业务/多市场）|
| 不用 | TTM PE（双重打折）、PB（成长股不适合）、PEG（G 不稳）、DCF（FCF 波动）|
| **个股案例** | 长城汽车（v3 P/S 主）|
| 加权规则 | P/S 50% + 中周期 PE 30% + SOTP 20% 加权中位 |

### 5. 高成长（PEG 或 P/S 主）

| 维度 | 表现 |
|------|------|
| 识别 | 营收增速 >25%、利润未释放或微利（医药/SaaS/早期新能源）|
| 主估值 | **PEG**（盈利已释放时）或 **P/S**（未盈利）|
| 辅 | SOTP、市场对标（IPO 价/比较公司）|
| 不用 | TTM PE、PB |
| **个股案例** | 绿的谐波（高 PE，需 PEG 校验）|

## P/S 推导通用模板（弱周期成长专用）

```
合理 P/S = 全球可比龙头基准 P/S
        ± 规模因子（小公司折价 -0.10~-0.20 / 巨头溢价 +0.10）
        ± 成长因子（出海+高端化等结构变化 +0.05~+0.15）
        ± 净利率因子（短期承压 -0.10 / 长期高于同行 +0.10）
        ± 治理 / 公司特有因子 ±0.05

合理市值 = 中周期营收（前瞻 1-2 年） × 合理 P/S
合理价格 = 合理市值 / 总股本
```

## 子行业 P/S 基准（实证校准）

> 数据来源：2025 年报 / 2026Q1 / 全球可比公司公开数据
> Caveat：P/S 受净利率水平、成长性、全球化程度影响，会随宏观波动；2-3 年应重新校准
> 仅列入数据完整、可信度高的子行业；其余子行业（消费电子代工/化工/电池等）样本不足，
> 遇到具体公司分析时再实证校准并补回本节

**A 股汽车基准（按梯队分档，每档宽度 ≤0.4x）**：
- **龙头**（比亚迪/长城/吉利 A）：**0.7-1.0x**
- **二三线**（长安等）：**0.4-0.6x**
- **新势力**（理想/小鹏）：**1.5-2.5x**（成长溢价，需重新校准）

> 不给统一区间（0.5-1.0x 跨度 0.5x 已踩"区间宽度 <0.5x"红线，分档后每档宽度 ≤0.4x 才有实用参考意义）。

### 其他子行业

**遇到具体公司分析时再校准并补回本节**。建议样本要求：
- 至少 3 家国内 A 股可比 + 1 家全球龙头基准
- 营收/市值数据来自最近 1 年内财报
- 区间宽度 < 0.5x 才有参考意义（否则区间太宽无效）

**已知不适用 P/S 的子行业**（直接归入其他类别）：
- 大宗化工 / 钢铁 / 煤炭 / 水泥 → 真周期股，PB 主
- 银行 / 保险 / 券商 → 金融，PB 主

**校准频率**：每年 Q1（年报披露后）重新校准一次同行可比 P/S。
