# 系统性风险防御框架

> 目标：把类似 1990 日本、2000 互联网泡沫、2008 金融危机、2015 A股、2018 美股这种**级别的系统性大跌**，从"-50%~-80% 重伤"压缩到"-20%~-30% 可控回撤"。**不追求精准择顶**——历史上没人能稳定做到——只追求"在被消灭之前系统性减少暴露"。

## 一、核心论断（决定后面所有规则）

复盘 1990 日本、2000 美国、2008 全球、2015 A股、2018 Q4 美股，**"系统性大跌"的发生需要两类条件叠加**：

| 条件 | 性质 | 说明 |
|---|---|---|
| **估值在历史高位** | 必要条件 | 单独成立可以撑很久（2017-2021 美股估值高但不崩）|
| **央行紧缩末期 + 信用周期收缩** | 充分条件 | 单独成立未必砸盘（2022 倒挂后 2023 反创新高）|

→ **只有当"贵"+"流动性收缩"叠加时**，估值泡沫才会从"贵"变"崩"。这决定了防御要采用**双因子**（估值 + 宏观），单因子假信号率太高。

## 二、三层防御（从永久到动态）

层级越靠前，越**不依赖信号判断**，越靠后越依赖准确性。**做不到第 2、3 层至少要有第 1 层**。

### L1 — 永久性结构防御（不依赖任何信号）

无论市场状态如何，永远遵守的硬约束。**这一层比择时更重要**——它让你在错过顶部的情况下也不会被消灭。

| 约束 | 阈值 | 理由 |
|---|---|---|
| **不加杠杆** | 杠杆 = 0 | 大跌中杠杆是致命因素（margin call → 强平 → 死亡螺旋）|
| **类现金/短债 buffer** | 仓位下限 15-25% | 任何时点都有"不会被波动消灭"的部分 + 可在底部加仓的弹药 |
| **单只个股上限** | ≤ 15% | 防止单股黑天鹅一次性击穿组合 |
| **单一行业上限** | ≤ 25-30% | 防止行业级别系统性风险（白酒/地产/教培这种级别）|
| **单一市场上限** | ≤ 70% | A 股 / 港股 / 美股至少做地域分散 |
| **黄金等尾部对冲** | 0-15% | 见 [黄金趋势跟踪](../macro/黄金趋势跟踪.md)，不强制持有但作为可选层 |

### L2 — 估值分位驱动减仓（信号 1，最实用）

**当估值进入历史高位分位时，按比例下调权益仓位**——不是清仓，是降低斜率。2000 和 2007 在估值上都给了 12-18 个月预警窗口，问题是绝大多数人不愿在还在涨时减。

#### 估值锚点选取

| 市场 | 主锚点 | 数据源 | 历史均值 | 警戒分位 |
|---|---|---|---|---|
| **美股** | Shiller CAPE（10 年通胀调整 PE）| multpl.com / Robert Shiller 数据库 | ~17x | 90 分位 ≈ 30x（2000 顶 44x / 2007 顶 27x）|
| **A 股** | 中证全指 / 沪深 300 PE-TTM 历史百分位 | 中证指数官网 / 万得 | 因子分位 | 80 分位 |
| **港股** | 恒指 PE-TTM 历史百分位 | 恒生指数公司 | 因子分位 | 80 分位 |
| **板块级** | 行业 PE 历史百分位 | 同上分行业 | — | 90 分位（板块更易极端化）|

> **为什么用分位而不是绝对 PE**：不同市场结构性 PE 不同（美股结构性高于 A 股），分位让"贵不贵"在每个市场内自洽。

#### 减仓阶梯（双因子触发，不单独由估值决定）

| 估值分位 | 单纯估值高（无宏观信号）| 估值高 + 宏观信号触发 |
|---|---|---|
| < 60 分位 | 维持 | 维持 |
| 60-80 分位 | 不加仓，停止边际买入 | 减权益 5-10% |
| 80-90 分位 | 减权益 5-10% | 减权益 15-20% |
| > 90 分位 | 减权益 15-20% | 减权益 25-35% + 增配类现金/黄金 |

> 关键约束：**单纯估值高不触发激烈减仓**（避免假阳性，2017-2021 持续高估但没崩）。激烈减仓必须有 L3 宏观/结构信号确认。

### L3 — 宏观 & 市场结构二次确认（信号 2 & 3，调节斜率）

#### 宏观先行指标（按可靠性排序）

| 指标 | 含义 | 警戒阈值 | 历史命中 |
|---|---|---|---|
| **美债 10y-2y 收益率曲线** | 短端 > 长端 = 紧缩末期 | **倒挂**（< 0）| 2000 / 2007 / 2019 / 2022（衰退命中率最高）|
| **美债 10y-3m 收益率曲线** | 同上更敏感 | **倒挂**（< 0）| 同上，纽约联储模型用此做衰退概率 |
| **高收益债信用利差（HY OAS）** | 信贷市场风险定价 | **快速走阔 > 200bp 在 3 个月内** | 2008（破 1500bp）/ 2020（破 1100bp）/ 2015 中油气债危机 |
| **央行政策周期拐点** | 加息末期 + 第一次降息 | **首次"被动降息"**（非预防式）| 2007 / 2019（短暂）/ 2008 雷曼前的紧急降息 |
| **私营信贷/GDP gap（BIS）** | 信贷扩张是否超长期趋势 | **gap > 10pp** | 1989 日本 / 2007 美国 / 2010s 中国房地产 |
| **PMI** | 制造业景气 | **连续 3 个月 < 50 且持续下行** | 辅助验证，不单独触发 |

> **数据源**：FRED（联储经济数据库）/ ICE BofA HY OAS / BIS Credit Gap 数据库 / 国家统计局 PMI / Caixin PMI。中国侧另用 [宏观仪表盘](../macro/宏观仪表盘.md) 持续跟踪。

#### 市场结构（情绪/泡沫温度计）

| 指标 | 阈值 | 数据源 |
|---|---|---|
| **IPO 数量 + 首日涨幅中位数** | 首日涨幅中位数 > 50% 且 IPO 数量月度新高 | Wind / SEC EDGAR |
| **融资余额 / 流通市值**（A 股）| > 历史 80 分位 | 同花顺 iFind / 中证金融 |
| **散户新开户数**（A 股）| 单月 > 200 万户且加速 | 中国结算月报 |
| **并购溢价中位数** | 标的 EBITDA 倍数 > 历史均值 + 1σ | 二级市场卖方研报汇总 |
| **个人投资者持仓股票比例**（美股）| AAII Sentiment > 50% bullish 持续 | AAII 调查 |
| **波动率指数 VIX（极低读数）** | 长期 < 12 表示极度无忧（顶部信号）| CBOE |

> **使用方式**：结构指标**只作为否决项**（confirmation）——不是单独触发减仓的依据，而是"估值分位高 + 宏观信号亮 + 结构指标也亮 → 加大减仓力度"的第三确认。

## 三、双因子触发表（实际操作规则）

写成可机械执行的规则，避免临场犹豫。

```
读取：估值分位 V%（主市场）/ 收益率曲线 YC / HY 利差 ΔS / 央行状态 P

权益基准仓位 = 70%（示例值，按个人风险偏好定）

减仓档位：
┌───────────────────────────────────────────────┐
│ 触发条件                              │ 权益仓位调整 │
├───────────────────────────────────────────────┤
│ V% > 80                              │ -5%       │
│ V% > 90                              │ -10%      │
│ YC 倒挂 > 3 个月                     │ -5%       │
│ ΔS 在 3 个月内走阔 > 200bp            │ -10%      │
│ P 进入"被动降息"模式                  │ -10%      │
│ BIS credit gap > 10pp                │ -5%       │
└───────────────────────────────────────────────┘

→ 同一时段多条触发可叠加，但单次减仓上限设为 -25%（避免一次性踩空）
→ 触发后每月复审，信号反转则按相反顺序回补
```

> **示例（假设性）**：若某时点 V% = 85（A 股全 A）+ 美债 10y-2y 倒挂 + HY OAS 快速走阔 → 触发 -5% + -5% + -10% = -20%，权益仓位从 70% → 50%，对应增配类现金 / 黄金 / 长债。

## 四、历史复盘（验证规则有效性）

| 危机 | 估值信号（提前 12-18 月）| 宏观信号（提前 6-18 月）| 结构信号 | 双因子模型是否预警 |
|---|---|---|---|---|
| **1990 日本** | 日经 PE > 60x 历史顶 | 央行 1989 急加息至 6%、地产信贷 gap > 30pp | 银座地价/股价齐疯 | ✅ 双因子均亮 |
| **2000 美国互联网** | Shiller CAPE 44x 历史顶 | 联储 1999-2000 加息 6 次 / YC 倒挂 | IPO 首日涨幅中位 90%+ | ✅ 三类信号齐亮 |
| **2007-08 全球金融** | 美股 CAPE 27x（90 分位）| YC 2006 倒挂、HY OAS 2007 中走阔 | 房贷违约率上升 / 房价收入比极端 | ✅ 双因子齐亮 |
| **2015 A 股** | 全 A PE 历史 90+ 分位 | 央行 2015-06 整治场外配资 | 融资余额 / 流通市值历史新高 / 月开户 400 万 | ✅ 估值 + 结构齐亮 |
| **2018 美股 Q4** | CAPE ~32x | 联储缩表 + 加息 4 次 / YC 接近倒挂 | — | ⚠️ 部分预警，回撤 -20% 后反弹 |
| **2020 新冠** | CAPE ~30x（80 分位）| 无紧缩信号 | — | ❌ **黑天鹅型**，规则失效 |

**重要观察**：

- **结构性大跌**（前 5 个）双因子模型都给了预警 → 规则可用
- **黑天鹅型**（2020 疫情、地震、战争）规则**失效** → 这是 L1 的工作（结构性 buffer + 黄金 + 单股上限），不能指望 L2/L3
- 2018 是规则的"假阳性边缘"——给了信号但没演变成系统性危机，**预警可能让你少赚但不会让你被消灭**，这是可接受的代价

## 五、关于"假信号"和心理纪律

**最大的难点不是识别信号，是在还在涨的时候执行。**

历史教训：
- 2007 年高盛 Strategist 已警告 → 大多数基金经理职业生涯压力下不敢减
- 2015 年 5 月已有 PE 分位警告 → "改革牛"叙事压制了减仓声音
- 2000 年 1 月 Buffett 在《财富》写文章警告 → 之后 14 个月 NASDAQ 还涨了 +50% 才崩

→ **规则必须机械化、写下来、提前承诺**。事后用情绪决策的人会一直错过。**接受"信号触发后市场可能再涨 6-12 个月"作为代价**，否则规则会被一次次推翻。

## 六、规则的局限（必须诚实承认）

1. **黑天鹅无法预防**（2020 疫情、2022 俄乌、地震、人为政策）→ 靠 L1
2. **信号假阳性**（2018、2022 都给过倒挂信号但未演变成 -50% 大跌）→ 接受少赚作为防御代价
3. **A 股特殊性**：政策驱动权重大于宏观（2014-2015 改革牛、2024-09 暴力反弹），单纯套用美股框架会失效
4. **"这次不一样"陷阱**：每轮泡沫顶部都有貌似合理的新叙事（互联网革命/AI/中国奇迹）→ 规则的价值正是**在叙事最有说服力的时候保持纪律**
5. **执行成本**：减仓后再回补会有税费/价差摩擦 → 在 A 股相对小（无资本利得税），美股较高

## 八、维护约定

- 双因子触发表的阈值**每年回顾一次**（每年 1 月），看是否需要根据上一年的表现调参
- 实际触发减仓的事件，**写入 [journal/](../journal/) 当日复盘**，记录：触发条件、执行减仓比例、事后 6/12/18 个月观察结果
- 信号给了但事后判断为假阳性的，记录在本页"历史复盘"表中，长期跟踪规则的精度
- 本框架与 [价值投资区间判定框架](价值投资区间判定框架.md) 双向同步：个股层换框架时，本框架的"L1 单股上限"等约束需复核

## 九、快速自检清单（每月跑一次）

```
[ ] 美债 10y-2y 是否倒挂？倒挂多久？
[ ] HY OAS 是否在 3 个月内走阔 > 200bp？
[ ] 美联储 / 央行是否进入"被动降息"模式？
[ ] Shiller CAPE 是否 > 30x（90 分位）？
[ ] A 股全 A / 沪深 300 PE 分位是否 > 80%？
[ ] 港股恒指 PE 分位是否 > 80%？
[ ] BIS 中国 / 美国 credit gap 是否 > 10pp？
[ ] PMI 是否连续 3 个月 < 50 且下行？
[ ] 融资余额 / 流通市值是否 > 历史 80 分位（A 股）？
[ ] IPO 首日涨幅中位数 / 月度数量是否极端（任一市场）？
[ ] 当前权益仓位是否符合双因子表？

→ 任意 ≥ 3 项触发，启动减仓档位计算
→ 任意 ≥ 5 项触发，启动减仓档位 + 写日志
```
